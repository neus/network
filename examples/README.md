# Examples

**Pick the path that matches how you build.** Each example shows how to obtain **`proofId`** (your **verification ID**) and run **`gateCheck`** when your environment is set up (credits, keys, wallet as required).

| Example | Best when you… |
| --- | --- |
| [nodejs-basic](./nodejs-basic/README.md) | Want server-side signing and full request control |
| [react-component](./react-component/README.md) | Want a browser gate with **`VerifyGate`** (Vite) |
| [curl-minimal](./curl-minimal/README.md) | Want raw HTTP or to debug signing end-to-end |
