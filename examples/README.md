# Examples

Start with the smallest example that matches what you are shipping.

| Example                                                        | Use it for                                          |
| -------------------------------------------------------------- | --------------------------------------------------- |
| [trust-receipts-showcase](./trust-receipts-showcase/README.md) | End-to-end product demo for reusable trust receipts |
| [react-component](./react-component/README.md)                 | Browser access with `VerifyGate`                    |
| [nodejs-basic](./nodejs-basic/README.md)                       | Server-side receipt creation and checks             |
| [curl-minimal](./curl-minimal/README.md)                       | Raw HTTP signing and debugging                      |
| [sdk-dev-chat-demo](./sdk-dev-chat-demo/README.md)             | Local SDK/widget development                        |

Default path: run the showcase locally, then use [Quickstart](https://docs.neus.network/quickstart) to add the same trust loop to your app.
