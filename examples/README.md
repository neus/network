# Examples

**Default demo (recommended):** [trust-receipts-showcase](./trust-receipts-showcase) — the front-facing open-source showcase for the SDK. Run it locally, then try the same ideas in the live product at [neus.network/demo](https://neus.network/demo).

**Pick the path that matches how you build.** Each example shows how to obtain **`qHash`** and run **`gateCheck`** when your environment is set up.

| Example | Best when you… |
| --- | --- |
| [trust-receipts-showcase](./trust-receipts-showcase/README.md) | Want the clearest real-world demo: every card maps a trust outcome to a verifier |
| [react-component](./react-component/README.md) | Want a browser gate with **`VerifyGate`** (Vite) |
| [nodejs-basic](./nodejs-basic/README.md) | Want server-side signing and full request control |
| [curl-minimal](./curl-minimal/README.md) | Want raw HTTP or to debug signing end-to-end |
| [sdk-dev-chat-demo](./sdk-dev-chat-demo/README.md) | Want a **minimal chat + ProofBadge** shell linked to `file:../../sdk` for widget and dev API testing |
