# OpenAI Apps SDK

## Connect NEUS Trust App

OpenAI Apps SDK builds on MCP. Remote MCP servers can power ChatGPT apps, deep research, and API integrations.

Use the hosted NEUS Trust endpoint in your OpenAI app configuration:

```
https://mcp.neus.network/mcp
```

Auth: OAuth 2.1 for human users; access keys for server integrations.

Full docs: [docs.neus.network/install](https://docs.neus.network/install)
