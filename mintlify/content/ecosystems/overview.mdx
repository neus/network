# Apps & Ecosystems

NEUS integrates with apps, protocols, and AI systems across the ecosystem.

## Integration Types

| Type | Description | Examples |
|------|-------------|----------|
| **Gate integrations** | Apps using VerifyGate for access control | NFT communities, DAOs, token-gated content |
| **API integrations** | Backend services using proof verification | KYC providers, risk analysis, compliance |
| **Agent integrations** | AI systems with verifiable identity | Trading bots, AI assistants, automation agents |
| **Hub integrators** | Organizations sponsoring verifications | DAOs, platforms, enterprise |

## Build on NEUS

### For App Developers

- **VerifyGate Widget**: Drop-in React component for gating
- **SDK**: TypeScript/JavaScript client for custom flows
- **Hosted Verify**: Offload auth UX to NEUS-hosted pages

### For Protocol Teams

- **Custom Verifiers**: Build new verification types on NEUS
- **Cross-chain Support**: EVM, Solana, and multi-chain identity
- **Privacy-first**: Private-by-default proof architecture

### For AI/Agent Builders

- **Agent Identity**: Register and verify AI agent credentials
- **Delegation**: Scoped permissions for autonomous operations
- **MCP Integration**: Model Context Protocol for AI tools
