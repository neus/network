The JavaScript SDK is the primary client surface for custom browser and app integrations. It wraps verification creation, deterministic signing helpers, hosted verify handoff, proof polling, and widget exports in one package.

## What the SDK Covers

<CardGroup cols={2}>
  <Card title="Create & Poll Proofs" icon="code" href="./javascript">
    Submit verification requests, standardize signing, and poll async proof status.
  </Card>
  <Card title="Authentication Helpers" icon="key" href="./authentication">
    Use hosted login and auth code exchange for simpler browser flows.
  </Card>
  <Card title="Verification Patterns" icon="shield" href="./verifications">
    Choose reuse, fresh proof, or gate-check strategies based on policy and UX needs.
  </Card>
  <Card title="Error Handling" icon="triangle-exclamation" href="./error-handling">
    Handle retries, hosted flow failures, interactive verifier handoff, and private proof auth.
  </Card>
</CardGroup>

## Install

```bash
npm install @neus/sdk
```

## Quick Start

```javascript
import { NeusClient } from '@neus/sdk';

const client = new NeusClient({
  apiUrl: 'https://api.neus.network',
  appId: 'acme-web'
});
```
