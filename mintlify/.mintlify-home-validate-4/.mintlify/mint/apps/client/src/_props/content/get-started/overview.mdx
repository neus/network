## Standardized surfaces

| Surface                        | URL                                  |
| ------------------------------ | ------------------------------------ |
| Hosted verify and hosted login | `https://neus.network/verify`        |
| Public docs                    | `https://docs.neus.network`          |
| Public API base                | `https://api.neus.network`           |
| Public verifier catalog        | `GET /api/v1/verification/verifiers` |
| Public gate check              | `GET /api/v1/proofs/check`           |

## Docs contract

* `docs.neus.network` is the canonical published surface.
* The repo-backed source content lives in this `mintlify/` tree.
* The public OpenAPI spec at `/openapi/public-api.json` is generated from `neus-nut` route metadata.
