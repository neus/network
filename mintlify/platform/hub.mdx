The Hub is the first-party UI at `https://neus.network/hub` for exploring proofs, managing identity, linking apps, and running verification flows.

## Explorer (Public)

The Explorer is public and does not require a wallet connection.

| Tab | Content |
|-----|---------|
| **Proofs** | Public proof feed with search, filters (verifier, tags, category), and sort (latest, trending 24h, trending 7d) |
| **Ecosystems** | Organizations and protocol metrics |
| **Agents** | Agent identities and delegation |
| **Creators** | Creator profiles and reputation |
| **Social** | Social feed |

URL params: `?view=<tab>`, `?q=<text>`, `?verifierId=<id>`, `?tags=<csv>`, `?sort=<latest|trending_24h|trending_7d>`, `?timeRange=<all|24h|7d|30d>`

## Account (Connected)

When you connect a wallet, the sidebar shows account tabs. All are available at `/profile?tab=<tab>`.

| Tab | Purpose |
|-----|---------|
| **Dashboard** | Overview, proof summary, quick actions |
| **Identity** | Domains (DNS TXT), organizations (Google/Microsoft OAuth), social accounts |
| **Agents** | Agent identity and delegation management |
| **Apps** | Link app domain, set daily credit cap |
| **Credits** | Balance, Pro upgrade, top-ups |
| **Activity** | Usage and history |
| **Settings** | Preferences |

## Identity

- **Domains:** Add a DNS TXT record at `_neus.<domain>` with your wallet address. Verify via Profile > Identity > Domains.
- **Organizations:** Prove membership via Google Workspace or Microsoft 365. OAuth flow is hosted; we do not store your email.
- **Privacy:** Choose public (discoverable) or private (owner-only) for each proof.

## Link App (Profile > Apps)

Link your integration domain to receive credits and attribution:

1. Enter app domain (e.g. `acme.com` or `https://acme.com`)
2. Set daily credit cap (default 10,000)
3. Use `*` for any origin only in development
4. Send `X-Neus-App` and `Origin` headers from your whitelisted domain

## Credits and Pro Upgrade

- **Credits:** View balance and usage in Profile > Credits.
- **Pro upgrade:** $99.99/month, 50,000 credits/month, card checkout. Upgrade from the Credits tab.
- **Top-ups:** Separate from subscription; managed in Credits tab.

## Proof Page

Each proof has a standard page at `https://neus.network/proof/[qHash]`:

- Share button
- Visibility badge (public/private/unlisted)
- Content hash and qHash
- Verifier and subject details
