## Stages

<Steps>
  <Step title="Start">
    Hosted `/verify`, SDK, or API.
  </Step>
  <Step title="Complete">
    Signature, hosted step, human flow, or on-chain lookup—per verifier.
  </Step>
  <Step title="NEUS verifies">
    Some instant; some async (poll status).
  </Step>
  <Step title="Receipt id">
    Save it—proof pages, checks, policy, profile.
  </Step>
</Steps>

## Modes

| Mode | When |
|------|------|
| Hosted `/verify` | Login, OAuth, human, org |
| SDK | Product-controlled flows |
| Raw HTTP | Server-only / advanced |
| Gate check | `GET /api/v1/proofs/check` — eligibility only |

## You get

- Response + status  
- Hosted handoff when needed  
- Poll endpoint when async  
- **Receipt id** to reuse
