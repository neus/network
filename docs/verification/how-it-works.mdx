## Stages

<Steps>
  <Step title="Start">
    Hosted `/verify`, SDK, or API.
  </Step>
  <Step title="Finish">
    Per verifier: sign, hosted step, lookup, etc.
  </Step>
  <Step title="Save id">
    Store **`proofId`** / **`qHash`** - [Verification results](../platform/receipts-and-results).
  </Step>
</Steps>

## Modes

| Mode | When |
|------|------|
| Hosted `/verify` | Login, OAuth, human, org |
| SDK | Product-controlled flows |
| Raw HTTP | Advanced / server-only |
| Gate check | `GET /api/v1/proofs/check` |

## You get

- Response + status
- **`hostedVerifyUrl`** when a browser step is required
- Poll when async
- **Verification id** to store
