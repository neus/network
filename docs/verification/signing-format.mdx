**Default:** hosted `/verify` or **SDK**. Raw `POST /api/v1/verification` only when you must.

## Why

The wallet agrees to the request; NEUS returns a **receipt**.

## HTTP

1. `POST /api/v1/verification/standardize` → `signerString`  
2. Sign it  
3. `POST /api/v1/verification` — **same body** + `signature`  

Only sign what `standardize` returns (or SDK `standardizeVerificationRequest`).

## Shape

UTF-8, **LF** newlines. Do not edit by hand.

```text
NEUS Verification Request
Wallet: <address>
Chain: <chainId or CAIP-2 chain>
Verifiers: <comma-separated-ids>
Data: <json payload>
Timestamp: <unix-ms>
```

## SDK

```javascript
import { standardizeVerificationRequest, signMessage } from '@neus/sdk';
```

Full example: [JavaScript SDK](../sdks/javascript)

## Wallets

| Type | Standard |
|------|----------|
| EOA | EIP-191 |
| Contract | EIP-1271 |
| Counterfactual | EIP-6492 |
