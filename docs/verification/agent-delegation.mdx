**Verifier ID:** `agent-delegation`

Use this verifier when a controller needs to grant limited authority to an agent.

## Best For

- payment delegation
- scoped actions
- auditable agent permissions

## Flow Type

Instant. The controller signs the delegation request.

## Required Fields

| Field | Required | Notes |
|------|----------|-------|
| `controllerWallet` or `controllerAccountId` | Yes | Controller identity; must match signer |
| `agentWallet` or `agentAccountId` | Yes | Agent identity |
| `agentId` | No | Reference to registered agent |
| `agentChainRef` | No | CAIP-2 chain reference when using `agentWallet` |
| `controllerChainRef` | No | CAIP-2 chain reference when using `controllerWallet` |
| `scope` | No | Example: `payments:x402` |
| `permissions` | No | Allowed actions |
| `expiresAt` | No | Unix ms expiry |

## Example

```json
{
  "controllerWallet": "0x...",
  "agentWallet": "0x...",
  "scope": "payments:x402"
}
```

This proof can be reused for agent policy checks, delegated actions, and payment-aware agent flows.
