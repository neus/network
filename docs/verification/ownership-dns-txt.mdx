**Verifier ID:** `ownership-dns-txt`

## Best For

- app onboarding
- org verification
- public trust pages for domains

## Flow Type

Lookup. NEUS checks the DNS TXT record after it is published.

## Required Fields

| Field | Required | Notes |
|------|----------|-------|
| `domain` | Yes | Hostname to verify |
| `walletAddress` | No | Optional override |

## Example

```json
{
  "domain": "example.com"
}
```
