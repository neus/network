Choose the verification capability that matches the trust decision you need to make.

## Choose by Outcome

| Required outcome | Verifier family |
|-------------------|------------------|
| Establish identity, association, or profile trust | Identity and Association |
| Prove ownership, provenance, or operator control | Ownership and Provenance |
| Gate access or monetize with trust checks | Access and Monetization |
| Reduce abuse or screen higher-risk actions | Abuse Reduction and Risk |
| Operate agents and delegated actions | Agent Operations |

## Verifier Modes

| Type | Completion | Use case |
|------|---------------|-----------|
| Instant | Signed and completed in one request | Fast wallet-based proof creation |
| Interactive | Completed in a hosted NEUS flow | Linked accounts, team verification, or human checks |
| Lookup | Checked against external or on-chain state | Live yes/no eligibility or ownership state |
