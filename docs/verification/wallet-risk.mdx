title: Wallet Safety Screening
description: Evaluate wallet risk before payments, access, or high-value actions.
icon: chart-line
---

**Verifier ID:** `wallet-risk`

## Best For

- payment flows
- fraud checks
- high-risk onboarding

## Flow Type

Lookup. NEUS checks wallet risk through an external provider.

## Required Fields

| Field | Required | Notes |
|------|----------|-------|
| `walletAddress` | Yes | Wallet to assess |
| `provider` | No | Example: `webacy` |
| `chain` or `chainId` | No | Optional chain context |

## Example

```json
{
  "walletAddress": "0x...",
  "chain": "base"
}
```
