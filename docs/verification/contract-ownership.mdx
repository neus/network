**Verifier ID:** `contract-ownership`

## Best For

- app ownership
- contract admin verification
- ecosystem and partner trust surfaces

## Flow Type

Lookup. NEUS checks owner or admin authority on-chain.

## Required Fields

| Field | Required | Notes |
|------|----------|-------|
| `contractAddress` | Yes | EVM contract address |
| `chainId` | Yes | EVM chain ID |
| `method` | No | `owner`, `admin`, or access-control style check |

## Example

```json
{
  "contractAddress": "0x...",
  "chainId": 8453
}
```
