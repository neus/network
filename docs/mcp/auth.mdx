**URL-only:** `neus_context`, catalog, verify links—no NEUS account on your balance. Order: [Overview](./overview).

**Bearer** ([Access keys](https://neus.network/profile?tab=account)): `neus_me`, private reads, calls that debit **your** credits. Same shape as `api.neus.network` ([SDK Authentication](../sdks/authentication)). Confirm with `neus_me` after connect.

Hosted `neus.network/verify` does **not** attach your IDE Bearer.

**Servers only:** trusted automation may reuse the same personal key on MCP HTTP. Never in browsers or mobile apps.

`neus_proofs_check` = yes/no only. Private proofs = API rules when signed in ([Proofs get](./proofs-get)).
