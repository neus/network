**Hosted MCP** works with **only the MCP URL** for catalog, context, hosted verification links, and other flows that do not require your NEUS account or billable usage on your balance.

Add **`Authorization: Bearer <token>`** when you need account-scoped tools—for example **`neus_me`**, **private proof reads**, or checks that debit **your** credits. Use the same token as the public API: create a **personal API key** on **[Access keys](https://neus.network/profile?tab=account)** (see [SDK Authentication](../sdks/authentication)). Call **`neus_me`** after connecting to confirm the session.

If you connect using only the MCP URL, tools can still return **`https://neus.network/verify`** links for browser completion. That browser step does not automatically attach your personal API key to your IDE MCP connection.

**Your app already has a NEUS session:** if the client supports headers, send the same **`Authorization: Bearer`** value you use for API requests.

**Privacy:** **`neus_proofs_check`** is yes/no only. Private proofs follow the same rules as the API when the user is signed in ([Proof context](./proofs-get)).
