| Step | Action |
|------|--------|
| 1 | Call **`neus_agent_link`** first when an agent wallet already exists |
| 2 | If setup is missing, call with **`agentId`** and **`agentWallet`** |
| 3 | Include **`controllerWallet`** only when you must set the approving account explicitly |
| 4 | Open a hosted URL only when the response returns one |
| 5 | Call **`neus_agent_link`** on the agent wallet to confirm readiness |

Vocabulary: [Agent concepts](../agents/concepts). **`agentWallet` is required**; generate and custody keys in your environment.

## Minimal input

```json
{ "agentId": "my-assistant", "agentWallet": "0x1234567890abcdef1234567890abcdef12345678" }
```

Delegation, spend limits, and production fields: [Agent delegation](../agents/agent-delegation) and the SDK. They are not required for the minimal create and confirm flow above.
