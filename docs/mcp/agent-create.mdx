1. Call with **`agentId`** and **`agentWallet`** (and **`controllerWallet`** if there is no NEUS session).
2. **Open** any hosted URL the response gives you.
3. **`neus_agent_link`** on the agent wallet to confirm readiness.

Vocabulary: [Agent concepts](../agents/concepts). **`agentWallet` is required** - generate and custody keys in your own environment.

## Minimal input

```json
{ "agentId": "my-assistant", "agentWallet": "0x1234567890abcdef1234567890abcdef12345678" }
```

## Optional delegation fields

- **`maxSpend`**: optional whole-number string in token base units (1 - 78 digits, no decimal point). For USDC (`payments:x402`), use six decimal places; convert display amounts with SDK `toAgentDelegationMaxSpend` before passing the value.
