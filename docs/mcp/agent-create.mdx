title: Agent Create (`neus_agent_create`)
description: Start agent setup; finish in hosted flow when prompted.
icon: robot
---

1. Call with **`agentId`** and **`agentWallet`** (and **`controllerWallet`** when you must supply the approver wallet explicitly—for example before hosted linking completes).
2. **Open** any hosted URL the response gives you.
3. **`neus_agent_link`** on the agent wallet to confirm readiness.

Vocabulary: [Agent concepts](../agents/concepts). **`agentWallet` is required** - generate and custody keys in your own environment.

## Minimal input

```json
{ "agentId": "my-assistant", "agentWallet": "0x1234567890abcdef1234567890abcdef12345678" }
```

Delegation, spend limits, and production agent fields: [Agent delegation](../agents/agent-delegation) and the SDK—not required for the minimal create → hosted flow above.
