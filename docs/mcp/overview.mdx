Native MCP support for agent integrations.

**Endpoint:** `https://mcp.neus.network/mcp`

---

## Quick Start

### 1. Add to Your MCP Client

**Cursor:**

```json
{
  "mcpServers": {
    "neus": {
      "url": "https://mcp.neus.network/mcp"
    }
  }
}
```

**Claude Desktop:**

```json
{
  "mcpServers": {
    "neus": {
      "command": "npx",
      "args": ["mcp-remote", "https://mcp.neus.network/mcp"]
    }
  }
}
```

### 2. Call the First Tool

```typescript
neus_verifiers_catalog()
```

### 3. Check Eligibility

```typescript
neus_verify_or_guide({
  "walletAddress": "0x...",
  "verifierIds": ["proof-of-human"]
})
```

---

## Available Tools

| Tool | Purpose |
|------|---------|
| `neus_verifiers_catalog` | List available verifiers |
| `neus_verify_or_guide` | Check + guide to hosted verify |
| `neus_proofs_check` | Eligibility check only |
| `neus_proofs_get` | Get user's proofs |
| `neus_me` | Session context |
| `neus_agent_link` | Check agent delegation |

---

## Decision Guide

| You Want To... | Use This Tool |
|----------------|---------------|
| Discover verifiers | `neus_verifiers_catalog` |
| Gate access + guide users | `neus_verify_or_guide` |
| Check eligibility only | `neus_proofs_check` |
| Get user context | `neus_me` |

---

## Standard Agent Loop

```
1. neus_verifiers_catalog()     — Discover capabilities
2. neus_verify_or_guide()       — Check or get hosted URL
3. User completes hosted flow   — Interactive verification
4. neus_proofs_check()          — Verify eligibility
```
