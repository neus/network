## Two URLs

| URL | Use |
|-----|-----|
| `https://mcp.neus.network/mcp` | MCP client transport for **`neus_*`** tools. |
| **`hostedVerifyUrl`** (from tools) | Browser verification. Open the URL from the tool response. |

## Recommended tool order

1. **Connect.** [Setup](./setup).
2. **`neus_context`.** Live verifiers, **`requiredFields`**, defaults (no key).
3. **Bearer.** **`neus_me`** → **`status: ok`**. Skip if you use public tools only ([Auth](./auth)).
4. **Work.** **`neus_proofs_check`**. **`neus_verify`** on instant verifiers when Bearer profile wallet matches **`walletAddress`**, or **`neus_verify_or_guide`** → **`hostedVerifyUrl`**. Agents: **`neus_agent_create`**, hosted steps, **`neus_agent_link`**.
5. **`neus_proofs_get`.** Proofs, profile summary, optional **`tags=`**, **`qHash`**.

Use **`neus_context`** for current **`requiredFields`**.

## Headless signing

**`neus_verify`** without **`signature`** (prepare) → sign **`signerString`** → **`neus_verify`** again with the **same** `walletAddress`, `verifierIds`, `data`, `signedTimestamp`, `chain`, plus **`signature`**. Do not repeat prepare.

Field names and tool list: [Tools](./tools). **`ownership-basic`:** [Ownership](../verification/ownership-basic). **402:** [x402](../platform/x402).
