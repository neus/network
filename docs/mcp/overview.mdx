**Endpoint alone:** public catalog, context, and hosted verification handoffs—**no API key**. **`Authorization: Bearer`** (personal key from **[Access keys](https://neus.network/profile?tab=account)**) adds **`neus_me`** and private proof reads. **[Auth](./auth)**

**[Get started](../get-started)** — **`appId`**, credits, **`sponsorGrant`** when your product pays for users. **[Billing](../platform/billing)** · **[Setup](./setup)**

`https://mcp.neus.network/mcp`

## Workflow

Add the MCP endpoint **once** in your client—see **[Setup](./setup)** for the configuration block.

1. **Browser verify:** assistant runs **`neus_verify_or_guide`** or **`neus_proofs_check`**. User opens **`https://neus.network/verify`** from the tool. Passkey or wallet stays in NEUS.
2. **Signed-in product:** same **`Authorization: Bearer`** as [SDK Authentication](../sdks/authentication). Then **`neus_me`**. **[Auth](./auth)**
3. **Proof history:** only if you already have an address or id. **`neus_proofs_get`**. [Tools](./tools)

| Goal | Tool |
|------|------|
| Send someone to browser verification | `neus_verify_or_guide` |
| Yes/no eligibility only | `neus_proofs_check` |
| Proof list + profile (you have an id) | `neus_proofs_get` |
| Signed-in “who am I” | `neus_me` · Bearer · [Auth](./auth) |
| Pick verifier IDs safely | `neus_verifiers_catalog` |
| Agents | `neus_agent_create`, `neus_agent_link` |

Open hosted URLs from tools.

## Agents

1. **`neus_agent_create`** → open any hosted URL returned → **`neus_agent_link`** to confirm.
