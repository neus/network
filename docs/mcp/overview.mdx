## Flow

1. `neus_context` (no key)
2. `neus_proofs_check` or `neus_verify_or_guide`
3. Open `hostedVerifyUrl` → `https://neus.network/verify`
4. `neus_proofs_get` when you have an id
5. Personal [access key](https://neus.network/profile?tab=account) for `neus_me` + private reads — [Auth](./auth)
6. `neus_verify`: one unsigned preparation call, then sign the **same** payload (do not repeat preparation)
7. Agents: `neus_agent_create` → hosted → `neus_agent_link`

HTTP **402**: [Billing](../platform/billing) · [x402](../platform/x402)

### Agent proofs vs MCP

`skills` on a proof are **metadata**—not MCP installs. The host supplies execution. Max **48** skill refs.

| Goal | Tool |
|------|------|
| Context | `neus_context` |
| Browser verify | `neus_verify_or_guide` |
| Eligibility | `neus_proofs_check` |
| List + profile | `neus_proofs_get` |
| Signed-in user | `neus_me` + access key |
| Pick ids | `neus_verifiers_catalog` |
| Agents | `neus_agent_create`, `neus_agent_link` |
