## Typical use

1. **`neus_context`** (no access key).
2. **`neus_proofs_check`** or **`neus_verify_or_guide`**; open **`hostedVerifyUrl`** when the tool returns it.
3. **`neus_proofs_get`** when you have a wallet or **verification ID**.
4. **Access key** from the hub only for **`neus_me`** and private reads - [Auth](./auth).
5. **Agents:** **`neus_agent_create`**, then **`neus_agent_link`**.
6. **`neus_verify`:** one unsigned prepare, then sign the **same** payload (do not call prepare twice).

HTTP **402:** same credits model as the rest of the API - [Billing](../platform/billing), [x402](../platform/x402).

### Agent proofs vs MCP

`skills` on a proof are **labels**, not installed MCP servers. The host runs tools. Max **48** skill refs.

| Goal | Tool |
|------|------|
| Context | `neus_context` |
| Browser verify | `neus_verify_or_guide` |
| Eligibility | `neus_proofs_check` |
| List + profile | `neus_proofs_get` |
| Signed-in user | `neus_me` + access key |
| Verifier ids | `neus_verifiers_catalog` |
| Agents | `neus_agent_create`, `neus_agent_link` |
