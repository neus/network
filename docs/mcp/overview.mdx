## Decision tree (call `neus_context` first every session)

| Step | Action |
|------|--------|
| 1 | **`neus_context`** — session home for profile and context |
| 2 | **`neus_proofs_check`** — check access **without creating** proof records |
| 3 | **`neus_verify_or_guide`** — hosted or guided path when proof state must be established in browser or provider |
| 4 | **`neus_verify`** — only when your Profile personal access key and signing path already satisfy NEUS for this verifier |
| 5 | **`neus_proofs_get`** — proof state, proof inventory, tags, optional delegated **`agentWallet`** reads |
| 6 | **`neus_agent_create`** → **`neus_agent_link`** — agent identity & delegation |

**Also:** **`neus_verifiers_catalog`** when **`neus_context`** is insufficient for schemas. **`neus_me`** once a Profile personal access key is configured on this MCP session (expect **`status: ok`**).

## Headless signing

| Step | What to send |
|------|--------------|
| Prepare | **`neus_verify`** without **`signature`**; sign **`signerString`** |
| Finish | **`neus_verify`** again with the same request fields plus **`signature`** |

Do not repeat prepare.
