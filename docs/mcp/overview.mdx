## Playbook

| Step | Action |
|------|--------|
| 1 | Connect with the hosted endpoint |
| 2 | Call **`neus_context`** for live verifiers and required fields |
| 3 | If using Bearer, confirm **`neus_me`** returns **`status: ok`** |
| 4 | Check or create proofs with **`neus_proofs_check`**, **`neus_verify`**, or **`neus_verify_or_guide`** |
| 5 | Open returned **`hostedVerifyUrl`** values for browser-only steps |
| 6 | Read proof state with **`neus_proofs_get`** |

## Headless signing

| Step | What to send |
|------|--------------|
| Prepare | **`neus_verify`** without **`signature`**; sign **`signerString`** |
| Finish | **`neus_verify`** again with the same request fields plus **`signature`** |

Do not repeat prepare.

Field names and tools: [Tools](./tools).
