Use the hosted NEUS MCP server when an AI client needs verifier lookup, gate checks, proof context, or a clean handoff into hosted verify.

## What MCP Gives You

- verifier catalog discovery
- eligibility-safe proof checks
- hosted verify guidance when human action is required
- public proof and profile context for wallets or DIDs

## Standard Agent Loop

1. `neus_verifiers_catalog`
2. `neus_proofs_check`
3. `neus_verify_or_guide`
4. `neus_proofs_get`
