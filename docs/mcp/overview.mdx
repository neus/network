## URLs

| URL | Role |
|-----|------|
| `https://mcp.neus.network/mcp` | MCP transport for **`neus_*`** tools |
| **`hostedVerifyUrl`**| Browser verification—open from the response |

## Playbook

| Step | Action |
|------|--------|
| 1 | Connect — [Setup](./setup) |
| 2 | **`neus_context`** — live verifiers, **`requiredFields`**, defaults (no key) |
| 3 | If using Bearer — **`neus_me`** → **`status: ok`** ([Auth](./auth)); skip if you only use public tools |
| 4 | Verify — **`neus_proofs_check`**; **`neus_verify`** (instant + matching profile wallet) or **`neus_verify_or_guide`** → **`hostedVerifyUrl`** |
| 5 | Agents — **`neus_agent_create`**, hosted steps, **`neus_agent_link`** |
| 6 | **`neus_proofs_get`** — proofs, profile summary, optional **`tags=`**, **`qHash`** |

Use **`neus_context`** for current **`requiredFields`**.

## Headless signing

| Step | What to send |
|------|----------------|
| Prepare | **`neus_verify`** without **`signature`** → sign **`signerString`** |
| Finish | **`neus_verify`** again with the **same** `walletAddress`, `verifierIds`, `data`, `signedTimestamp`, `chain`, plus **`signature`** |

Do not repeat prepare.

Field names and tools: [Tools](./tools). **`ownership-basic`:** [Ownership](../verification/ownership-basic). **402:** [x402](../platform/x402).
