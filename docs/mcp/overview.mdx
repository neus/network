**First call:** run **`neus_context`** after you connect. One response covers verifier context, billing model, and protocol rules. **No access key** required.

**Endpoint alone:** public catalog, context, and hosted verification handoffs. Add **`Authorization: Bearer`** with your **personal access key** from **[Access keys](https://neus.network/profile?tab=account)** when you need **`neus_me`**, private proof reads, or usage debited to **your** account. **[Auth](./auth)**

**[Get started](../get-started)** — create an app, choose who pays, generate an access key only when you need account-scoped tools. **[Billing](../platform/billing)** · **[Setup](./setup)**

`https://mcp.neus.network/mcp`

## Workflow

Add the MCP endpoint **once** in your client—see **[Setup](./setup)** for URL-only and authenticated configuration blocks.

1. **`neus_context`** — load SSOT context (public).
2. **Browser verify:** assistant runs **`neus_verify_or_guide`** or **`neus_proofs_check`**. User opens **`https://neus.network/verify`** from the tool. Passkey or wallet stays in NEUS (no pasted one-time codes in MCP).
3. **Signed-in assistant:** add **Bearer** with your **personal access key**, then **`neus_me`** when you need the current principal. **[Auth](./auth)**
4. **Proof history:** when you already have an address or id, **`neus_proofs_get`**. [Tools](./tools)

| Goal | Tool |
|------|------|
| Verifiers, billing model, protocol rules (call first) | `neus_context` |
| Send someone to browser verification | `neus_verify_or_guide` |
| Yes/no eligibility only | `neus_proofs_check` |
| Proof list + profile (you have an id) | `neus_proofs_get` |
| Signed-in “who am I” | `neus_me` · Bearer · [Auth](./auth) |
| Pick verifier IDs safely | `neus_verifiers_catalog` |
| Agents | `neus_agent_create`, `neus_agent_link` |

Open hosted URLs from tools.

## Agents

1. **`neus_agent_create`** → open any hosted URL returned → **`neus_agent_link`** to confirm.
