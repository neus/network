title: Public Proof Context (`neus_proofs_get`)
description: Load public proof context and profile summary for a wallet or DID.
icon: receipt
---

Use `neus_proofs_get` when you need public proof history and profile context for an agent workflow.

This tool is public-safe: it returns portable public context only and does not expose private proof payloads.

## Best For

- public proof context
- profile summarization
- agent-facing proof views

## Input

```json
{
  "identifier": "0xabc123...",
  "limit": 25,
  "offset": 0
}
```

## Stable Output

- resolved identifier
- public proof list
- profile summary when available
- pagination metadata

## Response Structure

```json
{
  "identifier": "0xabc123...",
  "resolvedAs": "address",
  "proofs": [
    {
      "proofId": "qhash...",
      "verifierId": "ownership-basic",
      "status": "verified",
      "createdAt": 1700000000000
    }
  ],
  "profile": {
    "displayName": "user.eth",
    "avatar": "https://..."
  },
  "pagination": {
    "total": 42,
    "limit": 25,
    "offset": 0,
    "hasMore": true
  }
}
```
