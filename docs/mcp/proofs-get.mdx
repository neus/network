Use when you **already have** an **`identifier`** (from NEUS, or user pasted it). If they still need to verify, **`neus_verify_or_guide`** + open the hosted link first.

**Visibility:** Without sign-in, you see what public browsing would show. With **Bearer** for that user, **private** proofs can appear ([Auth](./auth)).

## Input

```json
{
  "identifier": "0x…",
  "limit": 25,
  "offset": 0
}
```

`identifier`: EVM, Solana base58, or `did:pkh:…`.

## You get

Resolved wallet/DID, paginated **proofs**, **profileSummary**, optional **agentContext** / **contextPack**. Use the **`pagination`** object in the response as returned (including optional totals).
