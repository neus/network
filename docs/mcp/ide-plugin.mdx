Your assistant should know **who is real**, **what an agent is allowed to do**, and **which trust receipts already exist** — without re-proving identity every session.

**Endpoint:** `https://mcp.neus.network/mcp`

Signed in on [neus.network](https://neus.network/profile?tab=account)? Use **Open in** for Cursor, Claude Code, or Codex.

<Tabs sync={false}>
  <Tab title="Quick start">

npx -y -p @neus/sdk neus doctor --live

## Other editors

<AccordionGroup>
  <Accordion title="Claude Desktop" icon="desktop">

Add remote MCP URL **`https://mcp.neus.network/mcp`** in the app, or under `mcpServers` in `~/.claude.json`.

  </Accordion>

  <Accordion title="Windsurf" icon="wind">

Add to `~/.codeium/windsurf/mcp_config.json`:

```json
{
  "mcpServers": {
    "neus": {
      "url": "https://mcp.neus.network/mcp"
    }
  }
}
```

Restart Windsurf → Cascade → **MCPs**.

  </Accordion>

  <Accordion title="OpenClaw" icon="server">

`~/.openclaw/agents/main/agent/runtime-mcp.json`:

```json
{
  "mcpServers": {
    "neus": {
      "type": "http",
      "url": "https://mcp.neus.network/mcp",
      "headers": {
        "Authorization": "Bearer ${NEUS_ACCESS_KEY}"
      }
    }
  }
}
```

Set `NEUS_ACCESS_KEY` from [Profile → Account](https://neus.network/profile?tab=account).

Trust workflow skill:

```bash
cp -R plugins/neus-trust/skills/neus-trust-workflow ~/.openclaw/workspace/skills/
```

Clone [github.com/neus/network](https://github.com/neus/network) first.

  </Accordion>

  <Accordion title="Hermes" icon="bot">

`~/.hermes/config.json`:

  <Accordion title="Gemini CLI" icon="sparkles">

`~/.gemini/settings.json`:

{

## Servers & CI

```bash
npx -y -p @neus/sdk neus setup --access-key <npk_...>
```

Keys: [Profile → Account](https://neus.network/profile?tab=account). Never paste keys into chat.

## Upgrading from `neus-mcp`

```text
/plugin uninstall neus-mcp@neus
/plugin install neus-trust@neus
```
