## Config

Keep **`/mcp`** in the URL.

```json
{
  "mcpServers": {
    "neus": {
      "type": "streamableHttp",
      "url": "https://mcp.neus.network/mcp"
    }
  }
}
```

## Optional Bearer

`Authorization: Bearer` with **`npk_*`** matches REST. Enables **`neus_me`**, private reads, credits, and signatureless **`neus_verify`** when the profile wallet matches ([Auth](./auth)).

```json
{
  "mcpServers": {
    "neus": {
      "type": "streamableHttp",
      "url": "https://mcp.neus.network/mcp",
      "headers": { "Authorization": "Bearer <key>" }
    }
  }
}
```

| Check | |
|-------|--|
| Keys | [Access keys](https://neus.network/profile?tab=account) |
| Confirm | **`neus_me`** → **`status: ok`** |
| Never | Ship keys to the browser |

## First run

| # | Step |
|---|------|
| 1 | Connect (config above) |
| 2 | **`neus_context`** |
| 3 | If Bearer → **`neus_me`**; else skip |
| 4 | Verify, check, agents — [Tools](./tools) |
| 5 | **`neus_proofs_get`** when you need proofs or **`qHash`** |

Other clients: same URL, same optional **`Authorization`** header.
