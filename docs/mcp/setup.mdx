Shipping a product? [Get started](../get-started) first.

## URL only

Paste into MCP config (Cursor, Claude Code, Desktop, VS Code, …). Keep `/mcp` in the URL.

```json
{
  "mcpServers": {
    "neus": {
      "type": "streamableHttp",
      "url": "https://mcp.neus.network/mcp",
      "description": "NEUS MCP"
    }
  }
}
```

Reload if needed. URL-only is enough for `neus_context`, catalog, and verify links.

## First call

`neus_context` once after connect.

## Bearer

For `neus_me`, private reads, or **your** credits: `Authorization: Bearer` + [Access keys](https://neus.network/profile?tab=account). Same token as `api.neus.network`. [Auth](./auth)

Never in browser bundles. Prod apps: [Get started](../get-started), not your personal key.

### Cursor

`mcp.json`:

```json
{
  "mcpServers": {
    "neus": {
      "type": "streamableHttp",
      "url": "https://mcp.neus.network/mcp",
      "headers": { "Authorization": "Bearer <key>" }
    }
  }
}
```

### Claude Code

Same block in `.claude/mcp.json`.

### Other clients

Match your app’s **Streamable HTTP** docs. URL-only until headers are supported.

## First run

1. `neus_context`
2. `neus_verify_or_guide` or `neus_proofs_check`
3. Open `https://neus.network/verify` when given
4. `neus_proofs_get` if needed — [Tools](./tools)
5. Bearer for account — [Auth](./auth)

Then [Overview](./overview).
