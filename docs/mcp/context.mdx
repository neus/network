Call **`neus_context`** first on every MCP session—even when **no Profile access key** is wired in.

You receive a structured card with **`product`**, **`setup`** (hosted URLs + CLI snippets), **`mode`**, **`goldenPath`**, **`jobs`**, per-tool narration, **`proofModel`**, **`agentModel`**, **`safetyRules`**, and a compact **`verifierSummary`** (supplement with **`neus_verifiers_catalog`** when you truly need schemas).
