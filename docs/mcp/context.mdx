Call **`neus_context`** first after you connect to MCP. **No access key** required.

Use it to ground assistants in the current verifier set, integration constraints, and billing semantics before **`neus_verify_or_guide`**, **`neus_proofs_check`**, or **`neus_verifiers_catalog`**.
