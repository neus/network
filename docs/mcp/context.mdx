title: Context (`neus_context`)
description: NEUS MCP session home card—modes, URLs, verifier summaries, workflow.
icon: database
---

Call **`neus_context`** first on every MCP session—even when **no Profile access key** is wired in.

You receive a structured card with **`product`**, **`setup`** (endpoint + CLI snippets), **`mode`**, **`recommendedFlow`**, **`jobs`**, per-tool narration, **`proofModel`**, **`agentModel`**, **`safetyRules`**, and a compact **`verifierSummary`** (supplement with **`neus_verifiers_catalog`** when you need full schemas).

Recommended flow recap: **[Overview](./overview)**. Reuse existing receipts first; use hosted links only when the next step needs a browser.
