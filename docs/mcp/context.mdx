Call **`neus_context`** once at the start of a NEUS MCP session.

It returns the current mode, setup links, recommended flow, safety rules, compact verifier summary, and **signed-in profile context when the MCP connection is authenticated**.

Use **`neus_me`** only when you need to refresh profile context or look up a public wallet/DID.

Recommended flow recap: **[Overview](./overview)**. Reuse existing receipts first; use hosted links only when the next step needs a browser.
