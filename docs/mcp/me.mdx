---
title: Current Principal (`neus_me`)
description: Resolve the current principal, session-backed context, and agent associations.
icon: user-circle
---

Use `neus_me` when an MCP client needs to resolve the current principal or session-backed context.

With `NEUS_AUTH_TOKEN` set, the tool resolves the same NEUS session used by hosted login through `GET /api/v1/auth/me`.

## Tool Name

`neus_me`

## Best For

- current user context
- auth-required flows
- principal and agent association lookup

## Input

```json
{}
```

## What You Get Back

- the current principal
- session-backed identity context
- associated agent links when available

## Response Structure

```json
{
  "authenticated": true,
  "subjectId": "did:neus:...",
  "address": "0x...",
  "chainId": 8453,
  "agentLinks": [
    {
      "agentWallet": "0x...",
      "linked": true
    }
  ]
}
```
