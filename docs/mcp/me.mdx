---
title: Current user (`neus_me`)
description: Current account via MCP when you send a personal access key.
icon: user-circle
---

Call after **`neus_context`** when the connection sends **`Authorization: Bearer`** with a personal access key from [Access keys](https://neus.network/profile?tab=account) ([Auth](./auth), [Overview](./overview)). Same header as [API authentication](../api/authentication).

| Topic | Detail |
|-------|--------|
| **`neus_me` + Bearer** | End-user browser login is **not** passed through MCP |
| URL-only MCP | Returns **`auth_required`** |
| No key, user must verify | **`neus_verify_or_guide`** |
| Public identifier lookup | **`neus_proofs_get`** (visibility rules apply) |

## Input

```json
{}
```

## Output

| `status` | Payload |
|----------|---------|
| **`ok`** | `principal` (DID, handle, primary account), optional `agents` |
| **`auth_required`** | Optional **`ui_url`** |
