---
title: Current user (`neus_me`)
description: Current account via MCP when you send OAuth or a Profile access key.
icon: user-circle
---

Call after **`neus_context`** when the connection sends **`Authorization: Bearer`** — from **OAuth sign-in** or a **Profile access key** ([Access keys](https://neus.network/profile?tab=account)) ([Auth](./auth), [Overview](./overview)).

| Topic | Detail |
|-------|--------|
| **`neus_me` + Bearer** | End-user browser login is **not** passed through MCP |
| URL-only MCP | Returns **`auth_required`** — run **`neus auth`** or sign in through your host |
| No key, user must verify | **`neus_verify_or_guide`** |
| Public identifier lookup | **`neus_proofs_get`** (visibility rules apply) |

## Input

```json
{}
```

## Output

| `status` | Payload |
|----------|---------|
| **`ok`** | `principal` (DID, handle, primary account), optional `agents` |
| **`auth_required`** | Optional **`ui_url`** |
