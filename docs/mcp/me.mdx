---
title: Current user (`neus_me`)
description: Current account via MCP when you send a personal access key.
icon: user-circle
---

Call after **`neus_context`** when the MCP connection sends **`Authorization: Bearer`** with a personal access key from [Access keys](https://neus.network/profile?tab=account) ([Auth](./auth), [Overview](./overview)). Same header shape as [API authentication](../api/authentication). **`neus_me`** needs that Bearer—an end user’s interactive login in the browser is not passed through MCP. URL-only MCP returns **`auth_required`**.

End-user verification without your key: **`neus_verify_or_guide`**. Public identifier lookup: **`neus_proofs_get`** (visibility rules apply).

## Input

```json
{}
```

## Output

**`status: ok`:** `principal` (DID, handle, primary account), optional `agents`. **`status: auth_required`:** optional **`ui_url`**.
