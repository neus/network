Needs **`Authorization: Bearer`** on the MCP connection ([Auth](./auth)). **URL-only** MCP → **`auth_required`** (expected).

For “verify me” without a session: **`neus_verify_or_guide`** + hosted link; lookups: **`neus_proofs_get`** when you have an id.

## Input

```json
{}
```

## Output

**`ok`** — principal (DID, handle, wallet), optional **agents**. **`auth_required`** — may include **`ui_url`** to sign in.
