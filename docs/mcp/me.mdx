---
title: Current user (`neus_me`)
description: Signed-in NEUS user when MCP sends a session.
icon: user-circle
---

Needs **`Authorization: Bearer`** on the MCP connection ([Auth](./auth)). Use your **personal access key** from **[Access keys](https://neus.network/profile?tab=account)** as the Bearer token. If your MCP connection only includes the URL, the tool returns **`auth_required`** (expected).

For “verify me” without a session: **`neus_verify_or_guide`** + hosted link; public lookups: **`neus_proofs_get`** when you have an id (visibility rules apply).

## Input

```json
{}
```

## Output

**`ok`** - principal (DID, handle, wallet), optional **agents**. **`auth_required`** - may include **`ui_url`** to sign in.
