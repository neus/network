| URL | Use |
| --- | --- |
| **`https://mcp.neus.network/mcp`** | Production MCP (default). [Setup](./setup) |
| **`https://neus.network/verify`** | Hosted verification |
| **`https://mcp.neus.network/.well-known/mcp.json`** | MCP discovery descriptor |
| **`https://mcp.neus.network/.well-known/mcp/server-card.json`** | MCP server card for discovery consumers |
| **`https://dev.mcp.neus.network/mcp`** | Non-production MCP; pair with `https://dev.api.neus.network` only |
