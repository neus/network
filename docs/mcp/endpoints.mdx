The **MCP URL** and the **HTTP API** are different transports and base paths. Discovery and the [agent card](https://neus.network/.well-known/agent-card.json) list the endpoints your integration needs.

| URL | Role |
|-----|------|
| `https://mcp.neus.network/mcp` | MCP. [Setup](./setup) |
| `https://neus.network/verify` | Hosted verify |
| `https://mcp.neus.network/.well-known/mcp.json` | Discovery |
| `https://mcp.neus.network/.well-known/mcp/server-card.json` | Server card |
