## Public Endpoints

| Surface | Standard URL | Purpose |
| --- | --- | --- |
| Discovery | `https://mcp.neus.network/.well-known/mcp.json` | Server metadata and public tool listing |
| MCP transport | `https://mcp.neus.network/mcp` | Streamable HTTP MCP endpoint |
| Hosted verify | `https://neus.network/verify` | Wallet/passkey/browser fallback for interactive verification |

<Note>
`/mcp` is the standard public transport path. The server also accepts `/` in HTTP mode, but integrators should use `/mcp` so discovery metadata and client config stay consistent.
</Note>

## Supported Transports

### Hosted Streamable HTTP

This is the default go-live path for integrators.

```json
{
  "mcpServers": {
    "neus": {
      "type": "streamableHttp",
      "url": "https://mcp.neus.network/mcp"
    }
  }
}
```

### Stdio (Local)

Use stdio when you want the MCP process to run locally or inside your own infrastructure.

```json
{
  "mcpServers": {
    "neus": {
      "command": "npx",
      "args": ["-y", "@neus/mcp-server"],
      "env": {
        "NEUS_API_URL": "https://api.neus.network"
      }
    }
  }
}
```

## Health and Discovery (Hosted)

The hosted MCP at `mcp.neus.network` exposes:

| Endpoint | Purpose |
| --- | --- |
| `/.well-known/mcp.json` | Discovery metadata and tool listing |
| `/mcp` | Standard MCP transport |
| `/health` | Liveness probe |
| `/health/ready` | Readiness probe |
