---
title: Agent readiness (`neus_agent_link`)
description: Check whether an agent wallet has registered identity and approved access.
icon: robot
---

Check **`agentWallet`** for identity + access approval on file.

- **`linked: true`** → ready.
- **`link_required`** → open **`hostedVerifyUrl`** and follow **`nextSteps`**.

Optional **`principal`** (without session) points at the approving account. See [Agent concepts](../agents/concepts).

## Input

```json
{ "agentWallet": "0x…" }
```
