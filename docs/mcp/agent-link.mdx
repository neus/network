`neus_agent_link` is a **readiness check** for **`agentWallet`**: are identity and delegation already on file?

- **`linked: true`** (compat) → treat as **ready**.
- **`link_required`** → open **`hostedVerifyUrl`** / follow **`nextSteps`**.

Optional **`principal`** (without session) hints the **approving** account — not the agent wallet. Details: [Agent concepts](../agents/concepts).

## Input

```json
{ "agentWallet": "0x…" }
```
