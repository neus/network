---
title: Proof Check (`neus_proofs_check`)
description: Yes/no eligibility without proof details or hosted links.
icon: shield-check
---

**Yes/no only**—no proof IDs, no verify URL. For “open this link to verify,” use **`neus_verify_or_guide`**.

Private proofs count only with that user’s **Bearer** ([Auth](./auth)).

## Input

Uses **`wallet`** (not `walletAddress`):

```json
{
  "wallet": "0x…",
  "verifiers": ["ownership-basic"],
  "requireAll": true
}
```

## Output

`eligible`, `matchedCount`, `success`.
