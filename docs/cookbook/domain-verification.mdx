Verifier: [`ownership-dns-txt`](../verification/ownership-dns-txt). Full platform notes: [Platform setup](../platform/setup#dns-domain).

## Steps

1. **TXT** at `_neus.<yourdomain>`:

   ```text
   neus=<your-wallet-address>
   ```

   Example: `_neus.example.com` → `neus=0x1234…5678`

2. **Proof** (browser or server signing as for other direct-sign verifiers):

   ```javascript
   const proof = await client.verify({
     verifier: 'ownership-dns-txt',
     data: { domain: 'example.com', walletAddress: '0x...' },
     walletAddress: '0x...',
   });
   ```

3. **Check:**

   ```bash
   GET /api/v1/proofs/check?verifierIds=ownership-dns-txt&address=0x...
   ```

DNS can take minutes to propagate (sometimes longer per provider).
