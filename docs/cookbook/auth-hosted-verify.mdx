Ready-made verification flow. No frontend work required.

## When to Use Hosted Verify

| Use Case | Why Hosted |
|----------|------------|
| Login/signup | Guided wallet or passkey flow |
| Social linking | OAuth for Twitter, GitHub, Discord |
| Human verification | Interactive proof-of-human |
| Team verification | Org/workspace membership checks |

---

## How It Works

```
Your App                    NEUS Hosted                 User
   │                            │                         │
   │  1. Redirect to /verify    │                         │
   │───────────────────────────▶│                         │
   │                            │  2. User completes flow │
   │                            │◀────────────────────────│
   │  3. Return with proof ID   │                         │
   │◀───────────────────────────│                         │
   │                            │                         │
```

## Hosted URL

**Base:** `https://neus.network/verify`

| Parameter | Purpose |
|-----------|---------|
| `intent=login` | Session-first login flow |
| `verifiers=<ids>` | Comma-separated verifier IDs |
| `preset=<id>` | Pre-configured verifier bundle |
| `gateId=<id>` | Gate requirements |
| `returnUrl=<url>` | Where to redirect after completion |
| `mode=popup` | Popup flow with postMessage |

---

## Quick Start

### Redirect Flow

```javascript
const verifyUrl = new URL('https://neus.network/verify');
verifyUrl.searchParams.set('verifiers', 'ownership-basic');
verifyUrl.searchParams.set('returnUrl', 'https://myapp.com/callback');

window.location.href = verifyUrl.toString();
```

### Popup Flow

```javascript
const verifyUrl = new URL('https://neus.network/verify');
verifyUrl.searchParams.set('mode', 'popup');
verifyUrl.searchParams.set('origin', window.location.origin);

window.open(verifyUrl.toString(), 'neus-verify', 'width=500,height=700');

// Listen for completion
window.addEventListener('message', (event) => {
  if (event.data.type === 'neus_checkout_done') {
    console.log('Proof ID:', event.data.proofId);
  }
});
```

## Session Check

Use `/api/v1/auth/me` to check if a user is authenticated:

```bash
curl https://api.neus.network/api/v1/auth/me
```

| Response | Action |
|----------|--------|
| `authenticated: false` | Show login options |
| `authenticated: true` | User is verified |

---

## Completion Messages

For popup flows, the hosted window posts messages to the opener:

| Message | When |
|---------|------|
| `neus_login_complete` | Session confirmed |
| `neus_checkout_done` | Verification complete (includes `proofId`) |

## SDK Helper

```javascript
import { getHostedCheckoutUrl } from '@neus/sdk';

const url = getHostedCheckoutUrl({
  gateId: 'my-gate',
  returnUrl: 'https://myapp.com/callback',
  verifiers: ['ownership-basic'],
  intent: 'login',
});
```
