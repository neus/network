**Default:** users verify on NEUS; you store the **receipt id** and **check** before the next verify ([Integration](../integration#loop)). **Billing:** org credits + sponsors for most shipped apps ([Get started](../get-started#choose-who-pays)).

Apps: [Get started](../get-started) for `appId`, credits, sponsors.

Build links with the SDK (`returnUrl`). After redirect, finish session with `@neus/sdk`; use Bearer on API where needed ([SDK Authentication](../sdks/authentication)).

**MCP:** `neus_context` first; verify URLs work without a key. Bearer + [Access keys](https://neus.network/profile?tab=account) for account tools ([MCP Auth](../mcp/auth)).

## When

| Need | Why hosted |
|------|------------|
| Login | Guided signer |
| Social | OAuth in NEUS |
| Human / org | Interactive |

## URL

Base: `https://neus.network/verify`

| Param | Role |
|-------|------|
| `intent=login` | Session-first |
| `verifiers` | Comma-separated ids |
| `preset` | Bundled set |
| `gateId` | Gate requirements |
| `returnUrl` | Next page |
| `mode=popup` | Popup + `postMessage` |
| `origin` | Popup opener |

## Auth surfaces

Passkeys = WebAuthn in a **secure context**. Missing platform authenticator ≠ blocked cross-device / security-key.

| Surface | Passkeys | Wallet |
|--------|----------|--------|
| Desktop | Local, handoff, cross-device | Connect from page; QR needs a gesture |
| Mobile / PWA | Platform manager when available | Prefer wallet app / deep link |

Use **https** or **http://localhost** with an origin that matches your NEUS profile—not raw LAN IPs.

PWAs: don’t cache `/api/` auth traffic.

## Code

```javascript
import { getHostedCheckoutUrl } from '@neus/sdk';

window.location.href = getHostedCheckoutUrl({
  verifiers: ['ownership-basic'],
  returnUrl: 'https://myapp.com/callback',
});
```

**Popup:** `mode: 'popup'`, `origin: window.location.origin`, `window.open`, listen for `neus_checkout_done` / `neus_login_complete`. **Redirect:** handle `returnUrl` + SDK.
