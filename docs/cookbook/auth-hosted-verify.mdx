**Default:** user completes steps on NEUS; your app stores the **verification ID** and **checks** before asking again ([Integration](../integration#loop)). **Billing:** [Get started](../get-started) (app setup, org credits, **`sponsorGrant`** for customer traffic).

Build links with the SDK (`returnUrl`). After return, read the result with the SDK. Use an **access key** only when [SDK Authentication](../sdks/authentication) or [MCP Auth](../mcp/auth) says you need account scope.

**MCP:** `neus_context` first; verify URLs work without a key. Add a personal [access key](https://neus.network/profile?tab=account) for account tools ([MCP Auth](../mcp/auth)).

## When

| Need | Why hosted |
|------|------------|
| Login | Guided signer |
| Social | OAuth in NEUS |
| Human / org | Interactive |

## URL

Base: `https://neus.network/verify`

| Param | Role |
|-------|------|
| `intent=login` | Session-first |
| `verifiers` | Comma-separated ids |
| `preset` | Bundled set |
| `gateId` | Gate requirements |
| `returnUrl` | Next page |
| `mode=popup` | Popup + `postMessage` |
| `origin` | Popup opener |

## Auth surfaces

Passkeys = WebAuthn in a **secure context**. Missing platform authenticator is not blocked cross-device / security-key.

| Surface | Passkeys | Wallet |
|--------|----------|--------|
| Desktop | Local, handoff, cross-device | Connect from page; QR needs a gesture |
| Mobile / PWA | Platform manager when available | Prefer wallet app / deep link |

Use **https** or **http://localhost** with an origin that matches your NEUS profile - not raw LAN IPs.

PWAs: do not cache NEUS authentication responses in your service worker.

## Code

```javascript
import { getHostedCheckoutUrl } from '@neus/sdk';

window.location.href = getHostedCheckoutUrl({
  verifiers: ['ownership-basic'],
  returnUrl: 'https://myapp.com/callback',
});
```

**Popup:** `mode: 'popup'`, `origin: window.location.origin`, `window.open`, listen for `neus_checkout_done` / `neus_login_complete`. **Redirect:** handle `returnUrl` + SDK.
