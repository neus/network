`agent-identity` (agent wallet signs) and `agent-delegation` (approving account signs) are **separate** proofs. One person can do both, or roles can split. [Agent concepts](../agents/concepts).

## MCP

1. `neus_agent_create`
2. Open hosted URL from the response
3. `neus_agent_link`

## SDK

| Step | Page |
|------|------|
| Register | [Agent identity](../agents/agent-identity) |
| Delegate | [Agent delegation](../agents/agent-delegation) |
| Check | `GET /api/v1/proofs/check` · MCP tools |

Payload shapes match the JSON Schemas in [`docs/verifiers/schemas/`](https://github.com/neus/network/tree/main/docs/verifiers/schemas) (`agent-identity.json`, `agent-delegation.json`).
