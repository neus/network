| Layer | Verifier | When |
|-------|----------|------|
| Wallet | `ownership-basic` | Baseline uniqueness |
| Social | `ownership-social` | Linked account |
| Human | `proof-of-human` | Anti-bot |

## gateCheck examples

```javascript
const basic = await client.gateCheck({
  gateId: 'gate_basic',
  address: walletAddress,
});

const layered = await client.gateCheck({
  gateId: 'gate_layered',
  address: walletAddress,
});
```

## Hosted social

```jsx
<VerifyGate
  gateId="gate_social"
  onVerified={(result) => {
    // persist qHash for replay protection
  }}
>
  <button>Verify to continue</button>
</VerifyGate>
```

<Warning>
Configure recency on the gate for point-in-time checks. Store qHashes where replay matters.
</Warning>
