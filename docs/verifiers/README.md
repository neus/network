# Verifier reference (Markdown and JSON Schema)

This directory contains **field-level** Markdown pages and **JSON Schemas** (`schemas/`) for each verifier. They align with the verifier catalog returned by the NEUS API.

## Where to start

- **Guides (recommended):** [Verifier catalog](https://docs.neus.network/verification/verifiers) on the documentation site, then the per-verifier pages under **Build → Verification** (for example `/verification/ownership-basic`).
- **This folder:** use when you need schema-aligned field names, types, and copy-paste examples alongside the published guides.

Fetch the current verifier catalog at runtime: **`GET /api/v1/verification/verifiers`**.
