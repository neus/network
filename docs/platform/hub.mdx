The Hub is the public NEUS surface for profiles, proofs, and account management.

**URL:** `https://neus.network/hub`

---

## What Users See

| Tab | Purpose |
|-----|---------|
| **Dashboard** | Quick actions and proof summary |
| **Identity** | Domains, social proofs, and privacy settings |
| **Agents** | Agent registration and delegation |
| **Apps** | Domain linking and app attribution |
| **Credits** | Balance and top-ups |
| **Activity** | History and usage |

---

## Proof Pages

Every proof resolves to a public page:

```
https://neus.network/proof/[proof-id]
```

Shows:
- Proof receipt ID
- Verification type and timestamp
- Visibility state
- Share functionality

---

## Profile Discovery

- Browse public proof activity
- View verified identities
- Discover linked apps and agents
