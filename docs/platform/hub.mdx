The **NEUS platform** (sign in at `https://neus.network/hub`) is where you manage your profile, proofs, apps, and credits.

## What to complete before launch

1. Register each production domain used by your integration.
2. Set a daily credit cap that matches expected traffic.
3. Fund the wallet that should pay for verification requests.
4. Continue with the full onboarding checklist: **[Get started](../get-started)**.

Use your issued **`appId`** as **`X-Neus-App`** in API traffic. If you sponsor users, pair it with **`sponsorGrant`**. See **[Billing](./billing)** for payment models.

---

## What you can manage

| Area | Purpose |
|-----|---------|
| **Dashboard** | Account snapshot and recent activity |
| **Identity** | Domain and account proofs with privacy controls |
| **Agents** | Agent registration and delegation management |
| **Apps** | Domain-linked app attribution and limits |
| **Credits** | Balance management and top-ups |
| **Activity** | Request and verification history |

---

## Proof pages

Every proof resolves to a public page:

```
https://neus.network/proof/[proof-id]
```

Shows:
- Proof receipt ID
- Verification type and timestamp
- Visibility state
- Share functionality

- Browse public proof activity
