---
title: x402
description: Retry a billable request after payment.
icon: coins
---

When credits cannot cover a billable call, the API returns **402** with a quote. Pay, then retry with the receipt header.

## Flow

1. Call the API
2. **402** and quote
3. Pay
4. Retry with `PAYMENT-SIGNATURE: <receipt>`

## Header

```text
PAYMENT-SIGNATURE: <receipt>
```
