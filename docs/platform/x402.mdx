Default: **credits** + **sponsors** ([Get started](../get-started)).

**x402:** API returns **402** + quote → pay → retry with `PAYMENT-SIGNATURE`.

## When

|  |  |
|--|--|
| `POST /api/v1/verification` | No credits |
| `GET /api/v1/proofs/check` | No credits |
| MCP | No credits |
| Other billable calls | No credits on payer |

## Flow

1. Call API  
2. **402** + quote  
3. Pay  
4. Retry with `PAYMENT-SIGNATURE: <receipt>`

Same idea for verification, checks, hosted-adjacent calls, and agent flows that allow x402.

## Header

```text
PAYMENT-SIGNATURE: <receipt>
```
