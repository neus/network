NEUS bills **metered usage** (API, verification, related calls) to the **profile linked to your app**. **Credits** spend first. If a call is not covered, use **x402** ([x402](./x402)).

## What `appId` does

`appId` identifies your app on each request. It is **public** (not a secret). Create the app in [Platform setup](./setup). Pass the same value from the SDK or widgets.

## When credits run out

Billable calls return **402** with a quote. Complete **x402**. **Retry** with `PAYMENT-SIGNATURE`. Steps: [x402](./x402).

Optional visitor charges. [Payment gates](./payment-gates).
