## MCP tool order

1. `https://mcp.neus.network/mcp`. [Setup](../mcp/setup)
2. **`neus_context`**
3. Bearer, then **`neus_me`** (`status: ok`). [Auth](../mcp/auth)
4. **`neus_proofs_check`**, **`neus_verify`**, **`neus_verify_or_guide`**, agents. [Overview](../mcp/overview)
5. **`neus_proofs_get`**. [Proofs get](../mcp/proofs-get)

`npx -y -p @neus/sdk neus init` installs the hosted NEUS MCP server into supported clients automatically. Use `neus auth --access-key <npk_...>` when you want account-scoped tools such as `neus_me` and private reads.

## Rules

- Account, `appId`, and billing: [Get started](../get-started); detailed pricing in [Billing](../platform/billing)
- No in-app wallet: [Hosted Verify](../cookbook/auth-hosted-verify)
- No wallet **private keys** in MCP; **Access keys** only for Bearer account tools
- **402:** [x402](../platform/x402)
- Agents: [Agents](../agents/overview)
- Reuse **`proofId`**

## MCP JSON

**`type`** is client-specific; many accept **`streamableHttp`** for this URL.

```json
{
  "mcpServers": {
    "neus": {
      "type": "streamableHttp",
      "url": "https://mcp.neus.network/mcp"
    }
  }
}
```
