NEUS publishes documentation indexes for AI tools and editors. Fetch them when you want the site text available to a tool instead of a short summary.

## Route index (`llms.txt`)

`llms.txt` is a markdown index of page routes and descriptions:

```bash
curl https://docs.neus.network/llms.txt
```

## Full documentation (`llms-full.txt`)

`llms-full.txt` is the full documentation in one file:

```bash
curl https://docs.neus.network/llms-full.txt
```

## Cursor and MCP

For Cursor or other MCP-compatible editors, add the NEUS MCP server for live verification tools:

```json
{
  "mcpServers": {
    "neus": {
      "type": "streamableHttp",
      "url": "https://mcp.neus.network/mcp"
    }
  }
}
```

See [MCP Server](../mcp/overview) for tool reference and setup.
