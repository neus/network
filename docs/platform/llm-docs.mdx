## MCP flow

1. URL: `https://mcp.neus.network/mcp` — [Setup](../mcp/setup)  
2. `neus_context` first; then `neus_proofs_check` / `neus_verify_or_guide`; open `hostedVerifyUrl` when returned  
3. Bearer from [Access keys](https://neus.network/profile?tab=account) only for `neus_me` and vault reads — [Auth](../mcp/auth)  
4. `neus_verify` unsigned prep: reuse the same payload for the signed call (do not repeat `/standardize`)

**CLI:** `npx -y -p @neus/sdk neus init` — MCP JSON + URLs (stdout only).

## Assistant rules

- Billing / `appId`: [Get started](../get-started), [Billing](../platform/billing)  
- No wallet in-app: [Hosted Verify](../cookbook/auth-hosted-verify)  
- Never paste **wallet private keys** into MCP; Bearer keys are from Access keys only  
- x402: use when the client gets **402** and can retry with `PAYMENT-SIGNATURE` — [x402](../platform/x402)  
- Agents: [Agents overview](../agents/overview); align `appId` with Get started  
- Reuse saved `proofId` when possible
