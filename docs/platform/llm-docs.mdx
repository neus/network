## MCP

1. `https://mcp.neus.network/mcp` - [Setup](../mcp/setup) 
2. `neus_context` to check / verify-or-guide to open `hostedVerifyUrl` 
3. Personal [access key](https://neus.network/profile?tab=account) for `neus_me` / vault - [Auth](../mcp/auth) 
4. `neus_verify`: one preparation call; reuse the **same** payload when signing (do not repeat preparation)

CLI: `npx -y -p @neus/sdk neus init` (prints connection details to the terminal)

## Rules

- Billing / `appId`: [Get started](../get-started), [Billing](../platform/billing) 
- No in-app wallet: [Hosted Verify](../cookbook/auth-hosted-verify) 
- Never put **wallet private keys** in MCP; use **Access keys** (profile) for account tools only 
- Paid retries (**HTTP 402**): follow [x402](../platform/x402) 
- Agents: [Agents](../agents/overview) 
- Reuse saved `proofId`

## MCP JSON

Use the **`type` value your MCP client expects**; many clients accept `streamableHttp` for the hosted URL.

```json
{
  "mcpServers": {
    "neus": {
      "type": "streamableHttp",
      "url": "https://mcp.neus.network/mcp"
    }
  }
}
```
