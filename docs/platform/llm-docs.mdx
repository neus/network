## MCP

1. `https://mcp.neus.network/mcp` — [Setup](../mcp/setup)  
2. `neus_context` → check / verify-or-guide → open `hostedVerifyUrl`  
3. Bearer ([Access keys](https://neus.network/profile?tab=account)) for `neus_me` / vault — [Auth](../mcp/auth)  
4. `neus_verify`: one prep; same payload on signed call (no second `/standardize`)

CLI: `npx -y -p @neus/sdk neus init` (stdout only)

## Rules

- Billing / `appId`: [Get started](../get-started), [Billing](../platform/billing)  
- No in-app wallet: [Hosted Verify](../cookbook/auth-hosted-verify)  
- Never put **wallet private keys** in MCP; Bearer = Access keys only  
- HTTP **402**: `PAYMENT-SIGNATURE` — [x402](../platform/x402)  
- Agents: [Agents](../agents/overview)  
- Reuse saved `proofId`
