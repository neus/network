Credits pay for checks and saved results. Hosted sign-in is included. Credits apply when a check runs or a result is saved, not when someone signs in.

## Plans

| Plan | Price | Included |
|------|-------|----------|
| **Basic** | Free | 500 credits / month |
| **Pro** | $19.99 / mo · $199.90 / yr | 10,000 credits / month |
| **Enterprise** | Custom | Volume, SLAs, rollout support |

## How credits map to cost

Every action on NEUS costs **1 credit = $0.0025**. The difference is in how many credits an action requires:

- **Check an existing receipt (gate check):** 1 credit — **$0.0025** per check.
- **Create and save a new result:** 4–73 credits (**$0.01 to $0.18**) depending on the check type.

Reuse wins: creating a wallet-risk result costs $0.18. Checking it at any gate costs $0.0025. About 72× cheaper to reuse than to rebuild.

Live estimator: **[neus.network/pricing](https://neus.network/pricing)**. Heavier checks (human, agent authority, wallet risk) use more credits than a simple ownership check.
