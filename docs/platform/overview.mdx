NEUS connects completed identity, ownership, risk, and permission checks to the access, payment, and agent decisions that depend on them. Use hosted verification, reusable receipts, profiles, gates, API, SDK, widgets, or the hosted assistant connection without replacing your existing identity, wallet, payment, or agent stack.

Step-by-step onboarding: [Get started](../get-started). How it fits together: [Trust stack](./trust-stack).
