## Flow

```
┌─────────────────────────────────────────────────────────────┐
│                        NEUS Platform                         │
│                                                              │
│  ┌─────────────┐   ┌─────────────┐   ┌─────────────────┐   │
│  │   /verify   │   │  Proof API  │   │   /proof/[id]   │   │
│  │  (hosted)   │──▶│  (create)   │──▶│   (display)     │   │
│  └─────────────┘   └─────────────┘   └─────────────────┘   │
│         │                                      │            │
│         ▼                                      ▼            │
│  ┌─────────────┐                      ┌─────────────────┐   │
│  │  Platform   │                      │  Your App       │   │
│  │   `/hub`     │                      │  (reuse proofs) │   │
│  └─────────────┘                      └─────────────────┘   │
│                                                              │
└─────────────────────────────────────────────────────────────┘
```
