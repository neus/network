NEUS includes everything you need for verification. No infrastructure required.

## What You Get

| Component | What It Does |
|-----------|--------------|
| **Hosted Verify** | Ready-made verification UI at `/verify` |
| **Proof Pages** | Public page for each proof receipt |
| **Hub** | Profiles, discovery, and account management |
| **API** | Full programmatic access |
| **MCP** | Agent and AI tool integration |
