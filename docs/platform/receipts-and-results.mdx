## The Proof Receipt ID

When verification completes, you get back a **proof receipt ID**:

| Field | Where You See It |
|-------|------------------|
| `proofId` | SDK responses |
| `qHash` | HTTP paths and API |
| Proof receipt ID | User-facing docs |

**All refer to the same identifier.** Save it once. Reuse it everywhere.

---

## What to Save

| Save | Don't Save |
|------|------------|
| ✅ Proof receipt ID | ❌ Raw signatures |
| ✅ Verification status | ❌ Private proof payloads |
| ✅ Verifier IDs used | ❌ User credentials |

---

## What to Do With It

| Purpose | How |
|---------|-----|
| **Display** | Link to `/proof/[proof-id]` |
| **Gate** | Use `VerifyGate` widget |
| **Check** | Call `/api/v1/proofs/check` |
| **Verify** | Server-side eligibility check |

---

## Proof Page

Every proof has a public page:

```
https://neus.network/proof/[proof-id]
```

Shows:
- Proof receipt ID
- Verification type
- Timestamp
- Visibility (public/private/unlisted)

---

## Using the SDK

```javascript
// Create proof
const proof = await client.verify({ ... });
const proofId = proof.proofId;

// Check status
const record = await client.getProof(proofId);

// Gate check
const eligible = await client.gateCheck({
  address: '0x...',
  verifierIds: ['ownership-basic'],
});
```

---

## Using the API

```bash
# Proof record by receipt id
GET /api/v1/proofs/{qHash}

# Gate check
GET /api/v1/proofs/check?address=0x...&verifierIds=ownership-basic
```
