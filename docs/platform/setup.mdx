A **gate** is a public trust checkout entry in NEUS Hub. You choose the checks visitors must pass, publish, and embed checkout with one public `gateId`. No app ID, billing wallet, or API key for the default web path.

## Steps

1. Sign in at [neus.network/hub](https://neus.network/hub).
2. Open **Gates** in your profile and choose the checks visitors must pass.
3. Set pricing — you pay for checks by default, or charge visitors.
4. **Publish** and copy your **checkout link** or **embed snippet**.

Checks bill your NEUS account automatically because you own the gate. Visitors only need your `gateId`.

```jsx
import { VerifyGate } from '@neus/sdk/widgets';

<VerifyGate gateId="gate_abc123">
  <ProtectedContent />
</VerifyGate>
```
