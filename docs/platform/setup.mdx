In [Platform](https://neus.network/hub), link your product so requests carry **`appId`** and bill the right profile.

## Steps

1. **Apps** → create or link an app
2. Set **domain** and **daily credit cap**
3. Pass **`appId`** from the SDK or widgets (public, not a secret)

```javascript
import { NeusClient } from '@neus/sdk';

const client = new NeusClient({ appId: 'your-app-id' });
```
