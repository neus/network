Register your site in Hub before you ship **VerifyGate** or **Hosted Verify**. One setup step links your product, your domain, and your billing account.

## Steps

1. Sign in at [neus.network/hub](https://neus.network/hub).
2. Open **Apps** → **Register app**.
3. Enter your site URL and confirm once.
4. Copy your **app ID** (`appId`) and **billing wallet** (`billingWallet` — the wallet shown in Hub).

Pass both values in the SDK so user verifications spend **your** credits, not theirs.

```jsx
import { VerifyGate } from '@neus/sdk/widgets';

<VerifyGate
  appId="your-app-id"
  billingWallet="0xYourHubWallet"
  requiredVerifiers={['ownership-basic']}
>
  <ProtectedContent />
</VerifyGate>
```

## DNS domain verification

If you use [`ownership-dns-txt`](../verification/ownership-dns-txt) to prove control of a hostname, publish a **TXT** record on `_neus.<your-domain>` with value **`wallet=<your-wallet-or-account-address>`** (see the verifier page for Ethereum vs other networks, casing, and the optional `wallet=eip155:…` line).
