title: Rate Limits and Retries
description: 429s, backoff, and safe retries.
icon: clock
---

APIs may return **429** or transient errors.

- Exponential backoff; honor **Retry-After** when present.
- **Creates:** don’t blind-retry—confirm the first request didn’t succeed.
- **Polling:** a few seconds between polls; backoff on errors.
