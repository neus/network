## Script Tag Badge

```html
<script src="https://verify.neus.network/widget.js"></script>
<div data-neus-proof="0xYOUR_PROOF_RECEIPT_ID"></div>
```

The widget renders a badge for the provided proof receipt ID and links back to the standard proof page.

<Note>
The widget reads proof status from the public verification status route. Raw HTTP paths still use `:qHash` for the proof receipt ID parameter.
</Note>

## React Install

```bash
npm install @neus/sdk react react-dom
```

## ProofBadge

```jsx
import { ProofBadge } from '@neus/sdk/widgets';

```
