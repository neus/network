Drop-in React components. **`appId`** and optional **`sponsorGrant`** follow the same platform setup as the SDK — **[Get started](../get-started)**.

## Install

```bash
npm install @neus/sdk react react-dom
```

---

## Available Widgets

| Widget | Purpose |
|--------|---------|
| `VerifyGate` | Gate UI behind verification |
| `ProofBadge` | Display a proof receipt |

---

## VerifyGate

Gate content behind verification requirements:

```jsx
import { VerifyGate } from '@neus/sdk/widgets';

export default function ProtectedPage() {
  return (
    <VerifyGate
      requiredVerifiers={['nft-ownership']}
      verifierData={{
        'nft-ownership': {
          contractAddress: '0x...',
          tokenId: '1',
          chainId: 1,
        },
      }}
    >
      <ProtectedContent />
    </VerifyGate>
  );
}
```

---

## ProofBadge

Display a proof receipt on any page:

```jsx
import { ProofBadge } from '@neus/sdk/widgets';

<ProofBadge proofId="0x..." showChains />
```

---

## Script Tag (No React)

```html
<script src="https://verify.neus.network/widget.js"></script>
<div data-neus-proof="0xYOUR_PROOF_ID"></div>
```
