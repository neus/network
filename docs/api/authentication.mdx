**Default:** [Hosted Verify](../cookbook/auth-hosted-verify) or **SDK**. Raw HTTP only when intentional.

## Wallet standards

| Type | Standard | Wallets |
|------|----------|---------|
| EOA | EIP-191 | MetaMask, Coinbase, Rainbow |
| Contract | EIP-1271 | Safe, Argent, Sequence |
| Counterfactual | EIP-6492 | Undeployed accounts |

## HTTP create (advanced)

1. `POST /api/v1/verification/standardize` → `signerString`
2. Sign with user wallet
3. `POST /api/v1/verification` — same body + `signature`

Shapes: **API Reference** (OpenAPI). Story: [Signing format](../verification/signing-format)

## Picky wallets

Use `signMessage` from `@neus/sdk`—don’t hand-build payloads.
