**Base:** `https://api.neus.network`

**Billing:** same as SDK/MCP. [Get started](../get-started) · `appId` · `X-Neus-App` · `X-Sponsor-Grant`

Prefer **SDK**; use **HTTP** when you must.

## Endpoints

| Path | Method | Purpose |
|------|--------|---------|
| `/api/v1/verification` | POST | Create proof |
| `/api/v1/verification/standardize` | POST | Signing string |
| `/api/v1/proofs/{qHash}` | GET | Proof (`qHash` = SDK `proofId`) |
| `/api/v1/proofs/check` | GET | Eligibility |
| `/api/v1/proofs/by-wallet/{address}` | GET | List by wallet |
| `/api/v1/verification/verifiers` | GET | Catalog |

## Examples

**By wallet**

```bash
curl "https://api.neus.network/api/v1/proofs/by-wallet/0x...?limit=50"
```

Public / unlisted: anonymous ok. **Private:** user signed in.

**Check**

```bash
curl "https://api.neus.network/api/v1/proofs/check?address=0x...&verifierIds=ownership-basic"
```

**Create:** standardize → sign → POST `/verification`. Shapes: **API Reference** (OpenAPI).

**Catalog**

```bash
curl https://api.neus.network/api/v1/verification/verifiers
```

## Auth

- **Reads:** public/unlisted without login; private when owner session applies.
- **Writes:** wallet signature — [Authentication](./authentication)
