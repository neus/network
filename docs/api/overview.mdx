Use the API when your backend needs full control.

**Base URL:** `https://api.neus.network`

---

## Quick Reference

| Endpoint | Method | Purpose |
|----------|--------|---------|
| `/api/v1/verification` | POST | Create proof |
| `/api/v1/verification/standardize` | POST | Get signing string |
| `/api/v1/verification/status/{qHash}` | GET | Check status |
| `/api/v1/proofs/check` | GET | Gate check (eligibility only) |
| `/api/v1/proofs/by-wallet/{address}` | GET | Get proofs for wallet |
| `/api/v1/verification/verifiers` | GET | Verifier catalog |

---

## Proof Retrieval

### Get Proofs by Wallet

```bash
curl "https://api.neus.network/api/v1/proofs/by-wallet/0x...?limit=50"
```

**Owner-only params** (requires session or signature auth):
- `resolveContent=true` — Resolve private content for ownership-basic proofs
- `qHash=0x...` — Filter to single proof

Returns proofs with optional `resolvedContent` field when `resolveContent=true` and owner-authenticated.

## Create Proof

### 1. Get Signing String

```bash
curl -X POST https://api.neus.network/api/v1/verification/standardize \
  -H "Content-Type: application/json" \
  -d '{
    "walletAddress": "0x...",
    "verifierIds": ["ownership-basic"],
    "data": { "content": "Hello NEUS" }
  }'
```

Returns `signerString` — sign this with your wallet.

### 2. Submit Verification

```bash
curl -X POST https://api.neus.network/api/v1/verification \
  -H "Content-Type: application/json" \
  -d '{
    "walletAddress": "0x...",
    "signature": "0x...",
    "verifierIds": ["ownership-basic"],
    "data": { "content": "Hello NEUS" }
  }'
```

Returns `proofId` and `status`.

## Check Status

```bash
curl https://api.neus.network/api/v1/verification/status/{qHash}
```

---

## Gate Check

```bash
curl "https://api.neus.network/api/v1/proofs/check?address=0x...&verifierIds=ownership-basic"
```

Returns boolean eligibility for policy decisions.

---

## Verifier Catalog

```bash
curl https://api.neus.network/api/v1/verification/verifiers
```

Returns live verifier metadata.

---

## Authentication

- Read endpoints: Public
- Write endpoints: Require wallet signature
