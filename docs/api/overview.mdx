**Base:** `https://api.neus.network`

**Server integration?** Same payer rules as the SDK — complete **[Get started](../get-started)** (platform **`appId`**, **`X-Neus-App`**, **`X-Sponsor-Grant`**, credits).

Prefer the **SDK** when you can; use **HTTP** for direct server calls.

## Endpoints

| Path | Method | Purpose |
|------|--------|---------|
| `/api/v1/verification` | POST | Create proof |
| `/api/v1/verification/standardize` | POST | Signing string |
| `/api/v1/verification/status/{qHash}` | GET | Status |
| `/api/v1/proofs/check` | GET | Eligibility only |
| `/api/v1/proofs/by-wallet/{address}` | GET | Proofs for wallet |
| `/api/v1/verification/verifiers` | GET | Verifier catalog |

## Examples

**Proofs by wallet**

```bash
curl "https://api.neus.network/api/v1/proofs/by-wallet/0x…?limit=50"
```

Public and unlisted proofs are visible anonymously; **private** proofs need the user signed in.

**Gate check**

```bash
curl "https://api.neus.network/api/v1/proofs/check?address=0x…&verifierIds=ownership-basic"
```

**Create proof (outline)** — standardize → sign → POST `/verification`. Full request bodies: OpenAPI / SDK.

**Verifier catalog**

```bash
curl https://api.neus.network/api/v1/verification/verifiers
```

## Auth

- **Reads:** public/unlisted without login; private when the user is signed in.
- **Writes:** wallet signature (see [Authentication](./authentication)).
