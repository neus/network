## React: gate

```jsx
import { VerifyGate } from '@neus/sdk/widgets';

export default function ProtectedPage() {
  return (
    <VerifyGate
      appId="your-app-id"
      requiredVerifiers={['nft-ownership']}
      verifierData={{
        'nft-ownership': {
          contractAddress: '0x...',
          tokenId: '1',
          chainId: 8453,
        },
      }}
    >
      <ProtectedContent />
    </VerifyGate>
  );
}
```

## Node: proof

No wallet in `client.verify()` on the server → **standardize → sign → submit** ([JavaScript SDK](./sdks/javascript)) or [nodejs-basic](https://github.com/neus/network/tree/main/examples/nodejs-basic).

## HTTP (advanced)

```bash
curl -X POST https://api.neus.network/api/v1/verification/standardize \
  -H "Content-Type: application/json" \
  -d '{"walletAddress":"0x...","verifierIds":["ownership-basic"],"data":{"content":"Hello NEUS"}}'
# sign signerString, then POST /api/v1/verification with signature
```
