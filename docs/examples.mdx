These examples provide working code for one integration path at a time.

## Available Examples

| Example | Stack | What It Covers |
|---------|-------|----------------|
| [nodejs-basic](https://github.com/neus/network/tree/main/examples/nodejs-basic) | Node.js | Server-side verification and proof status checks |
| [react-component](https://github.com/neus/network/tree/main/examples/react-component) | React | `VerifyGate` integration for gated UI |
| [curl-minimal](https://github.com/neus/network/tree/main/examples/curl-minimal) | curl | Raw HTTP flow: standardize, sign, submit, check status |
