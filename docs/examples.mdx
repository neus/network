## React: gate

```jsx
import { VerifyGate } from '@neus/sdk/widgets';

export default function ProtectedPage() {
  return (
    <VerifyGate
      appId="your-app-id"
      requiredVerifiers={['nft-ownership']}
      verifierData={{
        'nft-ownership': {
          contractAddress: '0x...',
          tokenId: '1',
          chainId: 8453,
        },
      }}
    >
      <ProtectedContent />
    </VerifyGate>
  );
}
```

## Node: proof

**Server-held keys** or **HTTP-only:** [SDK](./sdks/javascript) or [nodejs-basic](https://github.com/neus/network/tree/main/examples/nodejs-basic) (`standardize` to sign to submit). Most apps should use **Hosted Verify** or **`client.verify()`** instead.

## HTTP (advanced)

```bash
curl -X POST https://api.neus.network/api/v1/verification/standardize \
  -H "Content-Type: application/json" \
  -d '{"walletAddress":"0x...","verifierIds":["ownership-basic"],"data":{"content":"Hello NEUS"}}'
# sign signerString, then POST /api/v1/verification with the same body + signature
```
