Use the NEUS brand kit when you need the logo, core colors, or short usage rules for product pages, integrations, or marketing materials.

## Logo

Use the primary NEUS logo on websites, apps, and docs. Keep it clear, readable, and on backgrounds with enough contrast.

| Asset | Use case | File |
|-------|----------|------|
| Primary logo | Web, apps, and docs | `/images/neus-logo.svg` |
| Monochrome variants | Print or constrained layouts | Derived from the primary asset |

## Color

| Token | Value | Use |
|-------|-------|-----|
| Primary accent | `#98C0EF` | Links, highlights, brand moments |
| Secondary accent | `#80A3DB` | Secondary emphasis |
| Dark background | `#101113` | Docs and product background |

## Usage Rules

- Do not stretch, rotate, or recolor the logo outside the brand palette.
- Do not place the logo on busy backgrounds that reduce legibility.
- Keep enough clear space around the mark so it reads as a single brand unit.
- Use the same logo and accent color across public docs, proof pages, and in-product UI.
