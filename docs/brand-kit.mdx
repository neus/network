Logo, palette, and quick rules for pages, apps, and partner materials.

## Logo

Generated in **this repo** (`npm run generate:docs-og`). Michroma wordmark + particle-ring mark on `#101113` canvas, 1200×630 PNGs. Favicon is copied read-only from the sibling `neus` brand pack.

| Asset | Path |
|-------|------|
| Favicon / tab / header | `/images/favicon.svg` |
| Default docs share card | `/images/og-default.png` |
| Section cards | `/images/og-quickstart.png`, `og-api.png`, `og-sdk.png`, `og-mcp.png`, `og-agents.png`, `og-verification.png`, `og-widgets.png` |

Product site OG (not duplicated here): `https://neus.network/images/neus-brand-pack/og-default-1200x630-neus.png`

Clear space, no stretch/rotate/off-palette recolor, readable contrast.

## Color

| Token | Hex | Use |
|-------|-----|-----|
| Accent | `#98C0EF` | Links, highlights |
| Accent 2 | `#80A3DB` | Secondary |
| Dark bg | `#101113` | Docs / product chrome |
