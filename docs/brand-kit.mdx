Use these assets so NEUS looks the same in docs, apps, and partner materials.

## Logo

One mark: `/images/neus-mark.svg` — full particle-ring depth in `#98C0EF` on `#101113`.

Header uses a trim-cropped supersampled PNG of that same mark (`neus-mark-256.png` at 48px) so Mintlify does not crush the dense SVG into a faint ring. Favicons use the SVG. Do not substitute a simplified or compact mark.

| Asset | Path |
|-------|------|
| Vector master | `/images/neus-mark.svg` |
| Header logo (resized) | `/images/neus-mark-256.png` |
| Favicon ICO (resized) | `/images/favicon.ico` |

Clear space, no stretch/rotate/off-palette recolor, readable contrast.

## Color

| Token | Hex | Use |
|-------|-----|-----|
| Accent | `#98C0EF` | Links, highlights |
| Accent 2 | `#80A3DB` | Secondary |
| Dark bg | `#101113` | Docs / product chrome |
