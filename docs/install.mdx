Connect your assistant to NEUS so it can check identity and permissions before sensitive actions. One setup works across Cursor, Claude Code, Codex, and VS Code.

Install once. Use the same profile and trust receipts across supported editors.

---

## Install

```bash
npm i -g @neus/sdk
neus setup
neus check
```

No global install? Run `npx -y -p @neus/sdk neus setup` once.

Ask your assistant:

```txt
Use NEUS Verify before taking sensitive actions.
```

Your assistant checks existing trust receipts first, guides any missing verification step, then summarizes the outcome as **NEUS Verify**: Passed, Action needed, or Blocked.

For example prompts, see [MCP journeys](./mcp/journeys).
