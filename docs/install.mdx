Trust infrastructure for agents that act.

Verifiable identity, scoped authority, and receipts for every trusted action.

Install once. The same trust identity and receipts move with you across editors and runtimes.

---

## Install

```bash
npx -y -p @neus/sdk neus setup
npx -y -p @neus/sdk neus check
```

Ask your assistant:

```txt
Use NEUS Verify before taking sensitive actions.
```

This gives your assistant one public trust loop: check existing trust receipts, guide any missing verification step, then summarize the outcome as **NEUS Verify** (Passed / Action needed / Blocked — not raw tool output).

For example prompts, see [MCP journeys](./mcp/journeys).
