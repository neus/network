Build a complete proof flow: verify, save the receipt, and reuse it.

## What You'll Build

By the end of this guide, you'll have:
- Created a verified proof using the SDK
- Saved a proof receipt ID
- Gated content behind verification

**Time:** Under 5 minutes

---

## 1. Install

```bash
npm install @neus/sdk
```

## 2. Create a Client

```javascript
import { NeusClient } from '@neus/sdk';

const client = new NeusClient({ appId: 'my-app' });
```

## 3. Create a Proof

The simplest proof binds a wallet to content:

```javascript
const proof = await client.verify({
  verifier: 'ownership-basic',
  content: 'My content or claim',
  wallet: window.ethereum,
});

// Save this ID — it's your reusable proof receipt
const proofId = proof.proofId;
```

## 4. Check Status

Most proofs complete instantly. Async verifiers return `202`:

```javascript
const status = await client.pollProofStatus(proofId, {
  interval: 3000,
  timeout: 60000,
});

console.log(status.status); // 'verified'
```

## 5. Gate Content

Use the proof receipt to gate React UI:

```jsx
import { VerifyGate } from '@neus/sdk/widgets';

<VerifyGate
  requiredVerifiers={['ownership-basic']}
>
  <ProtectedContent />
</VerifyGate>
```
