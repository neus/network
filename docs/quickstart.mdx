This walkthrough creates a first proof with **`ownership-basic`** and a browser wallet.

## 1. Install

```bash
npm install @neus/sdk
```

## 2. Create a proof

`client.verify()` defaults private.

```javascript
import { NeusClient } from '@neus/sdk';

const client = new NeusClient({
  appId: 'your-app-id',
});

const proof = await client.verify({
  verifier: 'ownership-basic',
  content: 'My content or claim',
  wallet: window.ethereum,
});

const proofId = proof.proofId;
```

Save **`proofId`**. Same value is **`qHash`** in some API paths. Check before asking the user to verify again.

## 3. Poll

```javascript
const status = await client.pollProofStatus(proofId, {
  interval: 3000,
  timeout: 60000,
});
```

## 4. Gate

```jsx
import { VerifyGate } from '@neus/sdk/widgets';

<VerifyGate
  appId="your-app-id"
  requiredVerifiers={['ownership-basic']}
>
  <ProtectedContent />
</VerifyGate>
```

## 5. Check from your server

Use **`gateCheck` in trusted server code** (Node, serverless route, backend worker), not in a public browser bundle, for security-sensitive allow/deny.

```javascript
const result = await client.gateCheck({
  address: '0x...',
  verifierIds: ['ownership-basic'],
});
```
