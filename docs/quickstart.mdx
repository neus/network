NEUS owns the standard browser ceremony. Your product saves the resulting receipt and checks it before trusted actions.

| Path         | Use it when                                    | Your app owns                                   |
| ------------ | ---------------------------------------------- | ----------------------------------------------- |
| `VerifyGate` | You want a React gate with a hosted fallback   | `appId`, required verifiers, protected UI       |
| Hosted URL   | You want redirect or popup control             | Return URL or popup listener                    |
| Server call  | Your backend creates receipts for linked users | `NEUS_ACCESS_KEY`, `verifyFromApp`, `gateCheck` |

## Browser

```jsx
import { VerifyGate } from "@neus/sdk/widgets";

<VerifyGate appId="your-app-id" requiredVerifiers={["ownership-basic"]}>
  <ProtectedContent />
</VerifyGate>;
```

Or send the user to Hosted Verify:

```javascript
import { getHostedCheckoutUrl } from "@neus/sdk";

window.location.assign(
  getHostedCheckoutUrl({
    verifiers: ["ownership-basic"],
    returnUrl: "https://myapp.com/auth/callback",
  }),
);
```

Hosted flow details: [Hosted Verify](./cookbook/auth-hosted-verify).

## Server

Keep access keys server-side only.

```env
NEUS_APP_ID=app_...
NEUS_ACCESS_KEY=npk_...
```

```javascript
import { NeusClient } from "@neus/sdk";

const client = new NeusClient({
  appId: process.env.NEUS_APP_ID,
  apiKey: process.env.NEUS_ACCESS_KEY,
});

await client.verifyFromApp({
  user: { walletAddress: user.walletAddress },
  verifier: "ownership-basic",
  content: "User completed onboarding"
);

const result = await client.gateCheck({
  address: user.walletAddress,
  verifierIds: ["ownership-basic"],
});
```

## Assistants and agents

Connect MCP hosts:

```bash
npx -y -p @neus/sdk neus setup
npx -y -p @neus/sdk neus auth
npx -y -p @neus/sdk neus doctor --live
```

Use **`neus auth`** for OAuth in your browser. Use **`--access-key`** only for servers and CI. Start with **`neus_context`**.

Bring an existing agent setup:

```bash
npx -y -p @neus/sdk neus import --dry-run
```

For agent wallets, follow [Agent verification flow](./agents/agent-verification-flow). Reuse `neus_agent_link` before `neus_agent_create`.
