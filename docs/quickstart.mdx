Direct-sign **`ownership-basic`**. Social / human / org → [Hosted Verify](./cookbook/auth-hosted-verify). Production → [Get started](./get-started).

## 1. Install

```bash
npm install @neus/sdk
```

## 2. Create a proof

`client.verify()` defaults **private** unless you set options. VerifyGate differs—[Widgets](./widgets/overview) · [Security and trust](./platform/security-and-trust).

```javascript
import { NeusClient } from '@neus/sdk';

const client = new NeusClient({
  appId: 'your-app-id', // optional locally; required in prod — [Get started](./get-started)
});

const proof = await client.verify({
  verifier: 'ownership-basic',
  content: 'My content or claim',
  wallet: window.ethereum,
});

const proofId = proof.proofId;
```

Persist `proofId` with the subject (same wallet passed to `verify`). The API may expose this as `qHash` — [Proof receipts](./platform/receipts-and-results). Reuse via `gateCheck` or VerifyGate.

More patterns: [SDK verifications](./sdks/verifications). No wallet in-page → [Hosted Verify](./cookbook/auth-hosted-verify).

## 3. Poll

```javascript
const status = await client.pollProofStatus(proofId, {
  interval: 3000,
  timeout: 60000,
});
```

## 4. Gate (React)

```jsx
import { VerifyGate } from '@neus/sdk/widgets';

<VerifyGate
  appId="your-app-id"
  requiredVerifiers={['ownership-basic']}
>
  <ProtectedContent />
</VerifyGate>
```

[VerifyGate](./widgets/verifygate) · [Integration](./integration)

## 5. Server check

- `gateCheck` **address** = identity that signed `verify()` (here, the `wallet` you passed).
- Prefer a **saved receipt id** + check before another verify.
- Verifier policy must match creation.

```javascript
const result = await client.gateCheck({
  address: '0x...',
  verifierIds: ['ownership-basic'],
});
```
