**~5 min:** install SDK → verify → save receipt ID → optional gate.

<Tip>
  **Shipping an app?** Do **[Get started](./get-started)** first (platform → Apps → credits → `appId` / sponsor). Then come back here for code.
</Tip>

## 1. Install

```bash
npm install @neus/sdk
```

## 2. Client + proof

```javascript
import { NeusClient } from '@neus/sdk';

const client = new NeusClient({ appId: 'my-app' });

const proof = await client.verify({
  verifier: 'ownership-basic',
  content: 'My content or claim',
  wallet: window.ethereum,
});

const proofId = proof.proofId; // save this
```

<Tip>
  **No in-app wallet?** Use [Hosted Verify](./cookbook/auth-hosted-verify) + session instead of `window.ethereum`.
</Tip>

## 3. Wait for completion

```javascript
const status = await client.pollProofStatus(proofId, {
  interval: 3000,
  timeout: 60000,
});
```

## 4. Optional: Gate in React

```jsx
import { VerifyGate } from '@neus/sdk/widgets';

<VerifyGate requiredVerifiers={['ownership-basic']}>
  <ProtectedContent />
</VerifyGate>
```
