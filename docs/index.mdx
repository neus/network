NEUS turns completed identity, ownership, risk, and permission checks into **trust receipts**. Connected apps and agents can check a receipt’s current status, scope, and expiration before access, payment, or another sensitive action.

**Verify once. Reuse the receipt.** Your NEUS **account** holds the receipts and agents you control. Visitors and agents do not rebuild the same checks in every product.

NEUS sits beside the auth, verification, payment, and agent stacks you already use. Those systems keep doing their jobs; NEUS carries an accepted result into the next gate.

## What NEUS handles

| Job | What you ship | Where to start |
|-----|---------------|----------------|
| Access without rebuilding KYC/OAuth | Hosted Verify + published gate | [Quickstart](./quickstart) · [Create a portal](./platform/setup) |
| Assistants that respect limits | MCP with hosted sign-in + verify-or-guide | [Install](./install) · [MCP journeys](./mcp/journeys) |
| Agents with scoped authority | Identity + permission receipts | [Agent verification](./agents/agent-verification-flow) |
| Monetize checks | You-pay credits, visitor checkout, or pay-per-call | [Pricing](./platform/pricing) · [Billing](./platform/billing) |

Free **Basic** includes monthly credits for hosted sign-in, portals, and API checks. **Pro** adds 50,000 monthly credits, higher API throughput, and priority support. [Plans](./platform/pricing).
