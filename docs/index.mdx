## The Core Loop

1. Verify a person, account, artifact, domain, team, asset, or agent.
2. Receive a proof receipt ID.
3. Reuse that proof receipt for access, profile trust, proof pages, and policy.
4. Send users back through hosted verify only when a fresh or interactive check is required.

## Public Surfaces

| Surface | What it gives you |
|---------|-------------------|
| `https://neus.network/verify` | Hosted verify for login and interactive checks |
| `https://neus.network/proof/[proof-receipt-id]` | Standard proof page for a saved proof receipt |
| `https://neus.network/hub` | Public profiles, proof discovery, apps, credits, and agents |
| `https://api.neus.network` | Public API for verification, status, and gate checks |
| `https://mcp.neus.network/mcp` | Hosted MCP endpoint for agent integrations |
