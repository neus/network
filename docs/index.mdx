## Core Product Surfaces

| Surface | What it gives you |
|---------|-------------------|
| `https://neus.network/verify` | Hosted login, checkout, and interactive verification |
| `https://neus.network/proof/[proof-receipt-id]` | Standard proof page for any proof receipt |
| `https://neus.network/profile` | Connected account dashboard and identity management |
| `https://neus.network/hub` | Discovery, proof feed, public profiles, and first-party explorer |
| `https://api.neus.network` | Public API and verifier catalog |
| `https://mcp.neus.network/mcp` | Hosted MCP endpoint for agent integrations |
