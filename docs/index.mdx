One proof receipt. Reusable across every app, API, and agent.

## What NEUS Does

Every product needs to verify users for login, access control, and trust. But each verification is typically:

- **Fragmented** вҖ” Built separately for each feature
- **Non-portable** вҖ” Can't reuse proofs across apps
- **Expensive** вҖ” Custom infrastructure per use case

**NEUS replaces fragmented verification with one reusable proof receipt.**

Verify once. Use that proof everywhere вҖ” login, gating, policy, and agent workflows.
