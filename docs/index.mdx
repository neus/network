NEUS records completed identity, ownership, risk, and permission checks as **trust receipts**. Connected systems can check a receipt’s current status, scope, and expiration before access, payment, or another sensitive action.

Create a receipt once, then reuse it where an integration accepts the same check.

Your **NEUS profile** shows the receipts and agents associated with your account. You control which supported receipts are public.

NEUS is a portable trust layer. It connects the authentication systems, verification providers, wallet tools, payment processors, policy engines, and agent runtimes you already use. Those systems keep doing their jobs; NEUS carries an accepted result into the next connected gate.
