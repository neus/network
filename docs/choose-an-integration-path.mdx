## Lowest friction first

| Situation | Start here |
|-----------|------------|
| Guided login or interactive verification without embedding a wallet in your app | **[Hosted Verify](./cookbook/auth-hosted-verify)** |
| AI tools (Cursor, Claude, VS Code, …) | **[MCP](./mcp/overview)** — add **`https://mcp.neus.network/mcp`** only; add **[Access keys](https://neus.network/profile?tab=account)** Bearer only if you need **`neus_me`** or private proof reads ([Auth](./mcp/auth)) |
| Production traffic or you are unsure who pays | **[Get started](./get-started)** before deep integration |
| Agent registration and delegation | **[Agents](./agents/overview)** after hub account and billing are clear |

## Paths

| Need | Path | Effort |
|------|------|--------|
| Guided login | [Hosted Verify](./cookbook/auth-hosted-verify) | Redirect or popup |
| React gates / badges | [Widgets](./widgets/overview) | `<VerifyGate>` |
| Custom code | [SDK](./sdks/overview) | Full control |
| Server checks | [API](./api/overview) | HTTP |
| AI (MCP) | [MCP](./mcp/overview) | `https://mcp.neus.network/mcp` |

**After you choose:** **[Integration](./integration)** — check → save → reuse for real traffic.
