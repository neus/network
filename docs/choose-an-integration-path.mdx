## Pick your path

| You need | Use | Effort |
|----------|-----|--------|
| Guided verification / login | [Hosted Verify](./cookbook/auth-hosted-verify) | Redirect or popup — no UI to build |
| React gating or badges | [Widgets](./widgets/overview) | One component — `<VerifyGate>` |
| Custom flow in code | [SDK](./sdks/overview) | Full programmatic control |
| Server-side checks | [API](./api/overview) | HTTP from any language |
| AI agents (Cursor, Claude) | [MCP](./mcp/overview) | One URL — `https://mcp.neus.network/mcp` |
