Match your stack to a row, then follow [Integration](./integration) for the check, save, and reuse loop.

| Need | Start here | Effort |
|------|------------|--------|
| Guided login or no embedded wallet | **[Hosted Verify](./cookbook/auth-hosted-verify)** | Redirect or popup |
| React gates / badges | **[Widgets](./widgets/overview)** | `<VerifyGate>` |
| Custom app code | **[SDK](./sdks/overview)** | Full control |
| AI tools (Cursor, Claude, VS Code, …) | **[MCP](./mcp/overview)** — endpoint only works without a key; **[Bearer](https://neus.network/profile?tab=account)** for **`neus_me`** / private reads ([Auth](./mcp/auth)) | Add URL once |
| Agents | **[Agents](./agents/overview)** (after hub + billing) | Presets + proofs |
| Raw HTTP (advanced) | **[API](./api/overview)** | Sign + POST |
| Unsure who pays | **[Get started](./get-started)** first | — |

**Then:** **[Integration](./integration)** — check → save → reuse.
