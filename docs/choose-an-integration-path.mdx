## Paths

| Need | Path | Effort |
|------|------|--------|
| Guided login | [Hosted Verify](./cookbook/auth-hosted-verify) | Redirect or popup |
| React gates / badges | [Widgets](./widgets/overview) | `<VerifyGate>` |
| Custom code | [SDK](./sdks/overview) | Full control |
| Server checks | [API](./api/overview) | HTTP |
| AI (MCP) | [MCP](./mcp/overview) | `https://mcp.neus.network/mcp` |

**After you choose:** **[Integration](./integration)** — check → save → reuse for real traffic.
