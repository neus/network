## Quick Decision

| Your Need | Use This | Why |
|-----------|----------|-----|
| Guided user verification | [Hosted Verify](./cookbook/auth-hosted-verify) | Zero frontend work |
| React gating or badges | [Widgets](./widgets/overview) | Drop-in components |
| Custom browser/server flow | [SDK](./sdks/overview) | Full control |
| Server-only checks | [API](./api/overview) | Backend ownership |
| AI agents and tools | [MCP](./mcp/overview) | Native agent support |
