## Full matrix

| Goal | Primary doc | Shape |
|------|-------------|-------|
| Universal identity (agents + linked proofs) | [Agents](./agents/overview) · [Wallet link](./verification/wallet-link) | Receipts + URLs + MCP |
| Login, social, human, org | [Hosted Verify](./cookbook/auth-hosted-verify) | Redirect or popup |
| In-wallet sign (`ownership-basic`) | [SDK](./sdks/overview) · [Quickstart](./quickstart) | Browser or server wallet |
| React | [Widgets](./widgets/overview) | VerifyGate |
| Server eligibility | [API](./api/overview), `gateCheck` | HTTP reads / checks |
| Assistants / IDE | [MCP](./mcp/overview) | MCP URL; access key for [account tools](./mcp/auth) |
| Raw HTTP proof create | [API](./api/overview) · [Authentication](./api/authentication) | Advanced: build the signed request body → sign → POST `/verification` |
| Who pays | [Get started](./get-started) | Credits, sponsors |

| Mode | When |
|------|------|
| Interactive | Hosted Verify or VerifyGate |
| In-browser wallet | SDK `verify` + `wallet` |
