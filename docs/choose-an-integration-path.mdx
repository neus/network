Choose **where** code runs: hosted checkout, browser SDK, raw HTTP, or MCP. The verification loop is the same everywhere ([Integration](./integration)); you need a NEUS account and linked app first ([Get started](./get-started)).

1. **Hosted checkout.** [Hosted Verify](./cookbook/auth-hosted-verify)
2. **Browser / React.** [SDK](./sdks/overview). [Quickstart](./quickstart). [VerifyGate](./widgets/overview)
3. **HTTP.** [API](./api/overview)
4. **Assistants / IDE.** [MCP](./mcp/overview)
