After [Get started](./get-started), choose a row, then [Integration](./integration).

| Goal | Primary doc | Shape |
|------|-------------|-------|
| Login, social, human, org | [Hosted Verify](./cookbook/auth-hosted-verify) | Redirect or popup |
| Direct-sign (`ownership-basic`) | [SDK](./sdks/overview) · [Quickstart](./quickstart) | Browser or server wallet |
| React | [Widgets](./widgets/overview) | VerifyGate |
| Server eligibility | [API](./api/overview), `gateCheck` | HTTP |
| Assistants / IDE | [MCP](./mcp/overview) | MCP URL; Bearer for [account tools](./mcp/auth) |
| Agents | [Agents](./agents/overview) | After app and billing |
| HTTP only | [API](./api/overview) | Standardize → sign → POST |
| Billing | [Get started](./get-started) | Credits, sponsors |

Interactive verifiers: Hosted Verify or VerifyGate. In-browser signing: SDK `verify` with `wallet`.
