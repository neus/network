## 1. Profile

1. **[Hub](https://neus.network/hub)** → sign in (passkey or wallet).
2. **Profile** = apps. **Credits** = balance. **Identity** = domains / org proofs when needed.

## 2. Link app

1. **`https://neus.network/profile?tab=apps`**
2. **Link app** or **Manage app**
3. **App domain** (e.g. `acme.com`)
4. **Daily credit cap**
5. Finish the modal (wallet sign if asked)

**`appId`** → **`X-Neus-App`**. SDK:

```javascript
import { NeusClient } from '@neus/sdk';

const client = new NeusClient({
  appId: 'your-app-id',
});
```

<Tip>
  **`appId`**: attribution only. **Pay for users:** **sponsor grant** (step 4).
</Tip>

## 3. Credits

1. **Credits** tab → balance on the **payer wallet**.
2. **Self-pay users:** credits on **their** wallet. **[Billing](./platform/billing)**

## 4. Sponsor grants

End users should not buy credits:

1. Org wallet funded (step 3).
2. **Short-lived** **`sponsorGrant`** → SDK → **`X-Sponsor-Grant`**.

```javascript
const client = new NeusClient({
  appId: 'your-app-id',
  sponsorGrant: '<sponsor-grant-token>',
});
```

Grants: **[Partner setup](./cookbook/hub-integrator)** · **[Billing](./platform/billing)**
