Add portable trust to your app, agent, or workflow with Hosted Verify, the NEUS API, or the SDK.

Select the integration that matches your product.

| Integration          | Use this when                                                                 | Start                       |
| -------------------- | ----------------------------------------------------------------------------- | --------------------------- |
| **App verification** | Your product needs reusable trust receipts before access, rewards, or payment | [Quickstart](./quickstart)  |
| **Agent trust**      | An agent needs identity, scoped authority, and receipts for trusted actions   | [Agents](./agents/overview) |
| **Assistant setup**  | Cursor, Claude Code, or Codex needs live identity and authority for agents | [NEUS for AI assistants](./mcp/ide-plugin) · [MCP setup](./mcp/setup) |

## App verification

1. Sign in at [neus.network/hub](https://neus.network/hub).
2. **Apps** → **Register app** — enter your site URL and confirm once.
3. Copy the **ready-made code snippet** Hub shows when registration finishes (or from **Apps** anytime). Paste it into your product — billing and app tagging are already filled in.
4. Save the receipt ID from verify callbacks and run **`gateCheck`** on your server before trusted actions.

Users complete checks on NEUS. They do not need NEUS accounts or API keys.

## Agent trust

1. `npx -y -p @neus/sdk neus import` (optional)
2. **`neus_context`** → **`neus_agent_link`**
3. **`neus_agent_create`** when proofs are missing — finish on NEUS
4. **`neus_agent_link`** until **`linked: true`**

Details: [Agent identity](./agents/agent-identity) · [Agent delegation](./agents/agent-delegation)

## Assistant setup

**CLI-supported hosts:** Cursor, VS Code/Codex, and Claude Code.

```bash
npx -y -p @neus/sdk neus setup
npx -y -p @neus/sdk neus auth
npx -y -p @neus/sdk neus doctor --live
```

Sign in with **`neus auth`** (OAuth). Access keys are for servers and CI only.

**Other MCP hosts** (Claude Desktop, Windsurf, OpenClaw, and more): [NEUS for AI assistants](./mcp/ide-plugin).

Claude Code marketplace: install **`neus-trust@neus`** from the same guide.
