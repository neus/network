Set up the account that owns your app, then ship one proof flow.

## 1. Sign in to NEUS

[Open NEUS](https://neus.network/hub) with passkey or wallet.

## 2. Link your app

Create the app in Platform and copy the **`appId`**. Requests from that app bill the linked profile.

## 3. Run your first loop

Use Hosted Verify for the browser handoff, or start with the SDK quickstart.
