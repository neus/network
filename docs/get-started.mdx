Pick the lane that matches what you are shipping.

| -------------------- | ----------------------------------------------------------------------------- | --------------------------- |

## App verification

1. Sign in at [neus.network/hub](https://neus.network/hub).
2. Create an app and copy your `appId`.
3. Add `VerifyGate`, a hosted verify URL, or a server-side receipt call.
4. Save the receipt ID and check it before trusted actions.

## Agent Trust

Import your existing local AI agent setups instantly, or link new agents to proof-backed receipts.

1. **Plug and Play Import:** Run `npx -y -p @neus/sdk neus import` to scan and import setups from OpenClaw, Cursor, Claude Code, or Claude Desktop.
2. **Register Identity:** Create `agent-identity` for the agent's wallet address.
3. **Delegate Scoped Authority:** Add `agent-delegation` when a controller user or organization defines what the agent is allowed to do.
4. **Link & Reuse:** Call `neus_agent_link` before actions to leverage pre-minted trust receipts.

## Assistant & MCP Setup

Connect Cursor, VS Code, Claude Code, or other MCP hosts to our hosted endpoint for live trust context:

```bash
npx -y -p @neus/sdk neus setup
npx -y -p @neus/sdk neus auth
npx -y -p @neus/sdk neus doctor --live
```

Sign in with **`neus auth`** (OAuth). Use **`neus setup --access-key <npk_...>`** only for servers and CI. Start each session with **`neus_context`**.

## Bring Your Own Agent (BYOA) in 30 Seconds

Already have an agent setup? Use our plug-and-play CLI to instantly import your existing rules, memories, and tools into the NEUS network:

```bash
npx -y -p @neus/sdk neus import
```
*Automatically detects and imports setups from OpenClaw, Cursor, Claude Code, and Claude Desktop, binding them to a secure, reusable trust receipt.*
