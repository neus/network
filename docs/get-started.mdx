## App verification

1. Sign in at [neus.network](https://neus.network) and open **Gates** in your profile.
2. Create a gate, choose the checks visitors must pass, and **Publish**.
3. Copy the **checkout link** or **`<VerifyGate gateId="..." />`** embed from the publish step.
4. Optionally run **`gateCheck({ gateId, address })`** on your server before trusted actions.

Users complete checks on NEUS. They do not need NEUS accounts or API keys. The published gate owns the checkout policy, verifier set, and default billing.

## Agent trust and assistants

```bash
npx -y -p @neus/sdk neus setup
npx -y -p @neus/sdk neus doctor --live
```

Setup covers Cursor, Codex, VS Code, and Claude Code. Then **`neus_context`** -> **`neus_agent_link`** -> **`neus_agent_create`** when proofs are missing.

Details: [MCP setup](./mcp/setup) - [Agent identity](./agents/agent-identity) - [Agent delegation](./agents/agent-delegation)

Codex only: `npx -y -p @neus/sdk neus setup --client codex`, then `npx -y -p @neus/sdk neus auth --client codex`.
