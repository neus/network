## 1. Account

1. [Hub](https://neus.network/hub)—passkey or wallet.
2. Finish profile so **Apps**, **Credits**, and **Access keys** are available.

## 2. First loop

Interactive flows → [Hosted Verify](./cookbook/auth-hosted-verify). First SDK sign → [Quickstart](./quickstart). Loop → [Integration](./integration).

## 3. Production setup

DNS, org OAuth, apps, sponsors at scale → [Platform setup](./platform/setup) after the loop is clear.

## Before launch

Decisions: **app**, **who pays**, **access key** (only for account-scoped API/MCP). Details: [Billing](./platform/billing) · [Platform setup](./platform/setup)

### Create an app

1. [Apps](https://neus.network/profile?tab=apps) → **Link app** / **Manage app**.
2. **App domain** + **daily credit cap**.

You get an `appId`; the SDK sends `X-Neus-App`.

```javascript
import { NeusClient } from '@neus/sdk';

const client = new NeusClient({
  appId: 'your-app-id',
});
```

`appId` is attribution, not billing. To pay for users: sponsor grants + org credits. [Billing](./platform/billing)

### Choose who pays

**Shipped products:** org credits + `sponsorGrant` (`X-Sponsor-Grant`). Users shouldn’t hit “buy credits” first. [Platform setup](./platform/setup)

**Self-serve:** user credits or x402—no sponsor on those flows.

```javascript
const client = new NeusClient({
  appId: 'your-app-id',
  sponsorGrant: '<sponsor-grant-token>',
});
```

### Access keys

[Access keys](https://neus.network/profile?tab=account) when you need `Authorization: Bearer` on `api.neus.network`, scripts, or MCP (e.g. `neus_me`). Wait until you see `auth_required`. [MCP Auth](./mcp/auth)

Never ship this key in the browser. Users use Hosted Verify, not your personal key.

### Launch checklist

- [ ] App + realistic daily cap
- [ ] Credits for your payer model
- [ ] `appId` always; `sponsorGrant` only when you sponsor
- [ ] Team read [Billing](./platform/billing) and [x402](./platform/x402)
- [ ] Receipt ids stored and reused ([Proof receipts](./platform/receipts-and-results))
