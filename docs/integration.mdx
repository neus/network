Login gets them in. Authority decides what they do next. Pick where you plug in.

## Where

| Place | Use |
| ----- | --- |
| Browser | Hosted Verify or VerifyGate |
| Server | SDK or API |
| Assistant | Install NEUS in your IDE |

Keep sign-in on NEUS hosted pages. Never put access keys in browser code.
