## Sequence

<Steps>
  <Step title="Register">
    `agent-identity` — agent wallet signs; one receipt for public metadata.
  </Step>
  <Step title="Delegate (if needed)">
    `agent-delegation` — approving account (`controllerWallet`) signs scope, caps, expiry.
  </Step>
  <Step title="Runtime">
    MCP or API: catalog, checks, hosted verify for interactive steps.
  </Step>
</Steps>

## MCP handoff

`neus_agent_link` → `status: "link_required"`: open **`hostedVerifyUrl`**, finish browser flow, call **`neus_agent_link`** again. Readiness for identity + permissions — not a generic wallet-link tool.
