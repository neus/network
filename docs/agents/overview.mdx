## Flow

1. **Register** — `agent-identity`  
2. **Delegate** — `agent-delegation`  
3. **Hosted** steps when needed; MCP uses the same verify URLs  

Steps 1–2 can be different wallets. [Concepts](./concepts)

## Why

| Need | How |
|------|-----|
| Which agent? | Registration binds wallet |
| What can it do? | Delegation scopes permissions |
| Product fit | MCP + receipts |

## Register

```javascript
await client.verify({
  verifier: 'agent-identity',
  data: {
    agentId: 'my-agent',
    agentWallet: agentWallet,
    agentLabel: 'My AI Agent',
    agentType: 'ai',
  },
  walletAddress: agentWallet,
});
```

## Delegate

```javascript
await client.verify({
  verifier: 'agent-delegation',
  data: {
    agentId: 'my-agent',
    controllerWallet: controllerWallet,
    agentWallet: agentWallet,
    permissions: ['read', 'write'],
  },
  walletAddress: controllerWallet,
});
```
