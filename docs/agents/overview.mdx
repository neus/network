Agent workflows issue proof receipts that can be reused across tools and automation.

## Why Teams Use Agent Flows

| Question | NEUS answer |
|----------|-------------|
| Which agent is acting? | Agent registration binds an agent to a wallet or account |
| What is it allowed to do? | Agent permissions define scope, permissions, and limits |
| How does it join real workflows? | Hosted verify, proof receipts, and MCP make the proofs usable |
