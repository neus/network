NEUS gives agent teams one reusable trust layer for identity, delegation, hosted verification, and MCP-based retrieval.

## Why Teams Use Agent Proofs

| Question | NEUS answer |
|----------|-------------|
| Which agent is acting? | `agent-identity` binds an agent to a wallet or account |
| What is it allowed to do? | `agent-delegation` defines scope, permissions, and limits |
| How does it join real workflows? | Hosted verify, proof reuse, and MCP make the proofs operational |
