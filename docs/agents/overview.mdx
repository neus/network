NEUS gives agents a portable identity and **scoped authority** that follows them across tools, IDEs, and apps. Other apps, agents, and payment rails can trust an agent before it acts, not after — because the agent carries its own proof.

| Need | How |
|------|-----|
| Identify the agent | `agent-identity` — the agent wallet signs for itself |
| Set what it can do | `agent-delegation` — a controller account grants scoped authority |
| Cap what it can spend | `maxSpend` on the delegation (base units) |
| Check if it is ready | `neus_agent_link` → `linked: true` |
| Be discoverable | A2A and EIP-8004 agent cards |
| Reuse trust | `qHash`, proof URL, or DID |
| Audit later | Receipt reads and proof pages |

Agent frameworks handle reasoning and execution. NEUS handles the trust layer they lack: verifiable identity, enforced scope, and receipts for every action.

## Two proofs make an agent trusted

- **Identity** (`agent-identity`) — the agent wallet self-attests its name, type, capabilities, and skills. Identity alone grants no authority.
- **Delegation** (`agent-delegation`) — a **controller** account signs to grant scoped authority to the agent: which actions it may take, spend caps, and an expiry.

An agent is **ready** only when both are on file.

## Scoped authority

A delegation is a hard boundary, not a hint:

- **`allowedActions` / `deniedActions`** — explicit allow/deny lists. A denied action always wins.
- **`maxSpend`** — a spend cap as a whole-number string in token base units (USDC uses 6 decimals: 25 USDC = `"25000000"`).
- **`scope`** — e.g. `payments:x402` for metered payments, or `global`.
- **`expiresAt`** — Unix milliseconds; set it whenever money or high-risk actions are in scope.

Quick setup presets (`full-access`, `payments`, `readonly`, `automation`) are available through `neus_agent_create`. Full field reference: [Agent delegation](./agent-delegation).

## Set it up

<Steps>
  <Step title="Connect">
    ```bash
    npx -y -p @neus/sdk neus setup
    npx -y -p @neus/sdk neus check
    ```
    Configures hosted NEUS MCP for Cursor, Codex, VS Code, or Claude Code.
  </Step>
  <Step title="Check">
    Call **`neus_context`**, then **`neus_agent_link`** with the agent wallet.
  </Step>
  <Step title="Create">
    If proofs are missing, call **`neus_agent_create`** — the agent wallet signs identity, the controller signs delegation. Finish on NEUS.
  </Step>
  <Step title="Confirm">
    Call **`neus_agent_link`** until **`linked: true`**, then reuse receipts.
  </Step>
  <Step title="Act">
    Check receipts before payments, data access, or other trusted actions.
  </Step>
</Steps>
