## Overview

**Register** the agent, then **grant** what it can do. These can be the same person or two different accounts.

## Roles

| Role | Action | Signs with |
|------|--------|------------|
| **Agent** | Registers agent metadata | Agent address |
| **Controller** | Grants scopes and limits | Controller account |

## Addresses

- **Agent address** - identifies the agent in NEUS.
- **Controller account** - signs delegation. Does not need to own the agent address.

## Signing

Use [hosted verify](../cookbook/auth-hosted-verify) or MCP. NEUS handles the ceremony.

## Field names

| Field | Meaning |
|-------|---------|
| `agentWallet` | Agent address |
| `controllerWallet` | Controller account |
| `linked` | Identity + delegation both present |
| `principal` | Signed-in account from MCP session |
