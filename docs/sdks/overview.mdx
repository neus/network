Install once, then use the SDK for browser verify, server checks, hosted verify URLs, and receipt reuse.

```bash
npm install @neus/sdk
```

**New app integration?** Start with [Get started](../get-started) before wiring the SDK.

## One flow

```javascript
import { NeusClient } from '@neus/sdk';
import { getHostedCheckoutUrl } from '@neus/sdk';

const client = new NeusClient({
  apiUrl: 'https://api.neus.network',
  appId: 'my-app',
});

const proof = await client.verify({
  verifier: 'ownership-basic',
  content: 'Hello NEUS',
  wallet: window.ethereum,
});
const proofId = proof.proofId;

await client.pollProofStatus(proofId, { interval: 3000, timeout: 60000 });

await client.gateCheck({
  address: '0x...',
  verifierIds: ['ownership-basic'],
});

const url = getHostedCheckoutUrl({
  verifiers: ['ownership-basic'],
  returnUrl: 'https://myapp.com/callback',
});
```

## Core methods

| Need | Use |
|------|-----|
| Create proof | `client.verify()` |
| Wait for async completion | `client.pollProofStatus()` |
| Eligibility | `client.gateCheck()` |
| Hosted redirect or popup | `getHostedCheckoutUrl()` |

Optional: `timeout` on the `NeusClient` constructor.

## Who pays for credits?

`appId` sets `X-Neus-App` for attribution. Credits follow the signer unless you add `sponsorGrant` or x402. See [Get started](../get-started) and [Billing](../platform/billing).
