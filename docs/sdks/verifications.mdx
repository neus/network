Default strategy: **`reuse-or-create`**.

## Privacy

`client.verify()` defaults to **private** stored results. VerifyGate create defaults to **unlisted public**. Full matrix: [Security and trust](../platform/security-and-trust)

| Option | When |
|--------|------|
| `privacyLevel: 'private'` | SDK default; owner or authorized access for private reads |
| `privacyLevel: 'public'` | Eligibility checks without owner auth; pair with `publicDisplay` |
| `publicDisplay: false` | Unlisted - still public to anyone with id |
| `publicDisplay: true` | Listed / discoverable (needs `public`) |

`storeOriginalContent: false` - hash-first when your verifier/policy allows.

Unlisted gate example:

```javascript
options: {
  privacyLevel: 'public',
  publicDisplay: false,
}
```

## Reuse-first

```jsx
<VerifyGate
  requiredVerifiers={['ownership-basic']}
  strategy="reuse-or-create"
>
  <Protected />
</VerifyGate>
```

## Fresh

High-stakes actions - force new proof:

```jsx
<VerifyGate strategy="fresh" requiredVerifiers={['ownership-basic']}>
  <Protected />
</VerifyGate>
```

## Read-only gate

```jsx
<VerifyGate strategy="reuse" requiredVerifiers={['ownership-basic']}>
  <Protected />
</VerifyGate>
```

## Hosted interactive

Hand off to `/verify` when the verifier needs it:

```jsx
<VerifyGate
  requiredVerifiers={['ownership-social']}
  hostedCheckoutUrl="https://neus.network/verify"
/>
```

[Hosted Verify](../cookbook/auth-hosted-verify)

## Server check

Prefer **`gateCheck`** over pulling full proofs for every decision:

```javascript
const result = await client.gateCheck({
  address: '0x...',
  verifierIds: ['token-holding'],
  contractAddress: '0x...',
  minBalance: '1',
  chainId: 8453,
});
```
