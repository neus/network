## Hosted login

1. Send users to NEUS hosted verify with your **`returnUrl`** — use **`getHostedCheckoutUrl`** ([Hosted Verify](../cookbook/auth-hosted-verify)).
2. They complete passkey or wallet on **neus.network**.
3. When they return to your app, **finish login with `@neus/sdk`** and keep the session token for API calls (**`Authorization: Bearer`**).

Same **Bearer** on MCP enables **`neus_me`** — [MCP Auth](../mcp/auth).

## Signature-based login

For flows that stay in your app without a hosted session:

```javascript
import { standardizeVerificationRequest, signMessage } from '@neus/sdk';

const standardized = await standardizeVerificationRequest(body, {
  apiUrl: 'https://api.neus.network',
});
const signature = await signMessage({
  provider: window.ethereum,
  walletAddress: body.walletAddress,
  message: standardized.signerString,
});
```

Use the SDK helpers instead of assembling signing strings yourself. Details: [Signing format](../verification/signing-format), [API authentication](../api/authentication).
