**Default:** no wallet or passkey code in **your** app — users verify on **`neus.network/verify`** via `VerifyGate` or `getHostedCheckoutUrl`.

| Where | What | Wallet code in your app |
|-------|------|---------------------------|
| Browser | `VerifyGate` + `appId` + `billingWallet` | No |
| Server | `NeusClient` + `gateCheck` with the same fields | No |
| Automation | `NeusClient` + `npk_*` access key | No |
| Advanced server | `verifyFromApp` after per-user approval | No |
| Browser (exception) | `client.verify({ wallet })` | Yes — [Signing format](../verification/signing-format) |

## VerifyGate (browser)

```jsx
import { VerifyGate } from '@neus/sdk/widgets';

<VerifyGate
  appId="your-app-id"
  billingWallet="0xYourHubWallet"
  requiredVerifiers={['ownership-basic']}
>
  <ProtectedContent />
</VerifyGate>;
```

## Hosted Verify (browser)

[Hosted Verify](../cookbook/auth-hosted-verify).

```javascript
import { getHostedCheckoutUrl } from '@neus/sdk';

const url = getHostedCheckoutUrl({
  verifiers: ['ownership-basic'],
  returnUrl: 'https://yourapp.com/auth/callback',
  appId: 'your-app-id',
  billingWallet: '0xYourHubWallet',
});
```

## Server reuse

```javascript
import { NeusClient } from '@neus/sdk';

const client = new NeusClient({
  appId: process.env.NEUS_APP_ID,
  billingWallet: process.env.NEUS_BILLING_WALLET,
});

await client.gateCheck({
  address: user.walletAddress,
  verifierIds: ['ownership-basic'],
});
```

## Personal access keys

`Authorization: Bearer <npk_...>` from [Access keys](https://neus.network/profile?tab=account). Server / MCP only.
