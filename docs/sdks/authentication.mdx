**Default:** no wallet or passkey code in **your** app — ceremony on **`neus.network/verify`** or server-only `verifyFromApp`.

| Where | What | Wallet code in your app |
|-------|------|---------------------------|
| Server | `NeusClient` + `verifyFromApp` / `gateCheck` | No |
| Browser | `VerifyGate` or `getHostedCheckoutUrl` → `/verify` | No |
| Browser (exception) | `client.verify({ wallet })` | Yes — [Signing format](../verification/signing-format) |

## App Link (server)

```javascript
import { NeusClient } from '@neus/sdk';

const client = new NeusClient({
  appId: process.env.NEUS_APP_ID,
  apiKey: process.env.NEUS_ACCESS_KEY,
});

const receipt = await client.verifyFromApp({
  user: { walletAddress: user.walletAddress },
  verifier: 'ownership-basic',
  content: { claim: 'User completed onboarding', source: 'your-app' },
});
```

## Hosted Verify (browser)

[Hosted Verify](../cookbook/auth-hosted-verify).

```javascript
import { getHostedCheckoutUrl } from '@neus/sdk';

const url = getHostedCheckoutUrl({
  verifiers: ['ownership-basic'],
  returnUrl: 'https://yourapp.com/auth/callback',
});
```

## Personal access keys

`Authorization: Bearer <npk_...>` from [Access keys](https://neus.network/profile?tab=account). Server / MCP only — never in browser bundles.
